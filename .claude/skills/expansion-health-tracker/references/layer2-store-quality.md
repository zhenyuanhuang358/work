# 第二层：单店质量层操作指南（T-1 同步指标）

核心问题：**扩张决策的结果，现在门店端已经显现了什么？**

数据来源：大众点评（评分 + 评论文本）

---

## 信号一：新店爬坡曲线

**拆解维度**：开业后第 1–6 个月的月度评分曲线和评论量变化。

### 数据采集

```
大众点评操作：
1. 搜索品牌名 → 门店列表 → 按"开业时间"排序
2. 筛选近 2 年内新开的门店（样本量：至少 20–30 家）
3. 每家门店查看"评价"tab → 按时间筛选开业前 6 个月的评论
4. 记录每月的：评论数量 + 大众评分（综合/口味/环境/服务）
5. 同样方法采集基准期（历史健康期，通常是 3–5 年前）的老店数据

WebSearch 补充：
[品牌名] 新店 开业 评价 口碑 [城市] [年份]
[品牌名] 大众点评 新店评分 开业初期
```

### 计算方法

```
爬坡曲线健康度 = 近2年新店均分（开业第6个月）- 基准期新店均分（同时段）

正常爬坡形态（健康基准）：
- 第1个月：评分偏高（新鲜感溢价，流量推广期）
- 第2–3个月：略降（真实消费者反馈涌入）
- 第4–6个月：企稳（稳定客群形成）
如果曲线持续下滑不企稳，是质量信号。
```

### 判断阈值

| 状态 | 判断标准 |
|------|---------|
| ✅ 健康 | 第6个月均分 ≥ 基准期 -0.2 分；评论量持续增长 |
| ⚠️ 警示 | 第6个月均分 -0.2 至 -0.5 分；评论量增速低于基准 |
| 🔴 危险 | 第6个月均分系统性低于基准 >0.5 分；**或曲线持续下滑不企稳** |

### 你的判断介入点

> 正常情况新店最初评分偏高（新鲜感溢价），第2–3个月略降后应该企稳。
> 系统性低于老店 = 开出来就不达标。
> 基准期应用该品牌自身的"健康期"数据（如2018–2019年），不是行业平均。

## 信号二：差评关键词结构

**拆解维度**：差评分类（服务/口味/性价比/等位）的占比变化，按季度追踪。

### 数据采集

```
大众点评操作：
1. 品牌门店 → 筛选"差评"（2星及以下）
2. 按时间分组：每季度抽取 50–100 条差评
3. 人工或 AI 分类（可用 Claude 批量分类）：
   - 服务类：服务态度、上菜速度、服务员问题
   - 口味类：食材质量、口味变化、食品安全
   - 性价比类：价格感知、量少价贵、优惠争议
   - 等位类：等待时间、预约问题、排队管理
4. 计算各类占比，追踪季度变化

AI 批量分类指令：
"以下是一批餐厅差评，请将每条分类为：服务/口味/性价比/等位/其他，
输出格式：原文 | 分类 | 一句关键词。共[N]条。"
```

### 计算方法

```
差评结构偏移指数 = 当季度某类占比 - 历史均值占比

各类正常占比参考（成熟品牌健康期）：
服务类：20–30%
口味类：25–35%
性价比类：15–25%
等位类：15–25%
```

### 判断阈值（每季度变化）

| 状态 | 判断标准 |
|------|---------|
| ✅ 健康 | 各类占比季度变化 < 5ppt；无单类持续上升趋势 |
| ⚠️ 警示 | 任一类别季度变化 5–10ppt；或连续 2 季度同向变化 |
| 🔴 危险 | 单类变化 > 10ppt；或服务类 > 45%，口味类 > 40% |

### 反直觉解读（核心差异化，必须应用）

| 差评关键词上升类型 | 表面现象 | **真实原因（你的解读）** |
|----------------|---------|---------------------|
| 服务类差评↑ | 服务质量变差 | **扩张过快，培训跟不上门店增速** |
| 口味类差评↑ | 食材或口味问题 | **供应链压力：门店数超出供应链有效覆盖半径** |
| 性价比类差评↑ | 消费者觉得贵 | **消费者用餐体验/期望失衡，是需求疲软预警** |
| 等位时间评论↓ | 运营效率提升 | **客流量减少，不是效率改善，门店冷清** |

## 信号三：新老店评分剪刀差

**拆解维度**：近 2 年新店 vs 5 年以上老店，同城评分差值。

### 数据采集

```
大众点评操作：
1. 选取目标城市（建议：北京/上海/成都/深圳各一个样本城市）
2. 筛选该城市品牌门店：按开业时间分组
   - "新店组"：近 2 年内开业
   - "老店组"：5 年以上开业
3. 记录各组的：综合评分均值 + 评论总数
4. 计算同城剪刀差 = 老店组均分 - 新店组均分
```

### 计算方法

```
剪刀差 = 老店组均分 - 新店组均分

正常情况：新店略低于老店（新鲜感 vs 稳定客群的区别），
健康差距：0–0.2 分，是正常的
系统性差距：>0.5 分，说明新店开出来就普遍低于品牌标准
```

### 判断阈值

| 状态 | 剪刀差 | 判断含义 |
|------|-------|---------|
| ✅ 健康 | 0–0.2 分 | 新老店质量一致，扩张标准化良好 |
| ⚠️ 警示 | 0.2–0.5 分 | 新店质量有压力，扩张速度接近供给上限 |
| 🔴 危险 | > 0.5 分 | **新店系统性不达标，是扩张失序的门店端确认信号** |

### 你的判断介入点

> 正常情况新店略低于老店（新鲜感期 vs 稳定客群），但差距应该在 0.2 分以内。
> 剪刀差 > 0.5 = 新店开出来就不达标，不是时间问题，是标准化问题。
> 这个信号通常比翻台率下滑早 1–2 个季度出现，是真正意义上的领先指标。
