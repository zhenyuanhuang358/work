# 扫描平台列表与搜索方法

每次扫描时，按优先级顺序搜索。优先覆盖中文平台（主要受众），再补充国际视角。

---

**第一轮（核心，必搜）**：微博热搜 / 知乎热榜 / 抖音热点 / 百度热搜 / X（英文互联网）

**第二轮（补充，选搜）**：36Kr / 华尔街见闻 / B站热榜 / 小红书 / Reddit

**第三轮（深挖，按需）**：Product Hunt / Hacker News / GitHub Trending / 专项财经媒体

## 搜索策略建议

**时间过滤**：搜索时加入今日日期，避免抓到过期内容。

**情绪信号词**（加在搜索词后面，提高命中率）：
```
爆了 / 炸了 / 怒了 / 刷屏 / 全网 / 又一个 / 没想到 / 打脸 / 翻车 / 崩了
```

**英文搜索补充词**：
```
broke / viral / everyone is talking / massive / outrage / shocked
```

**发现跨平台话题的信号**：同一事件在微博 + 知乎 + 抖音同时出现 → 高度爆发信号。
