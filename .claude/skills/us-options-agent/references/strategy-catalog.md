# 期权策略目录

选择策略前先对照账户参数：单笔风险 ≤ $425，总敞口 ≤ $2,550。

---

## 一、稳定权利金策略（占比 70%）

### Cash Secured Put（CSP）
- **逻辑**：卖 Put，愿意以行权价买入标的
- **理想条件**：IV Rank > 40，标的有支撑，中性偏多
- **风险**：标的暴跌，最大亏损 = 行权价 × 100 - 权利金
- **小账户适配**：选 ETF 或低价股，行权价控制保证金需求
- **止损**：权利金亏损达 2× 收取权利金时平仓

### Covered Call（CC）
- **逻辑**：持有正股 + 卖 Call，增加收益
- **理想条件**：持仓成本较低，IV 偏高，短期中性偏多
- **风险**：标的大涨，上行收益被封顶
- **小账户适配**：持仓成本 ≤ $3,000/100股

### Wheel Strategy
- **逻辑**：CSP → 被行权拿股 → CC → 被行权出股 → 循环
- **理想条件**：选愿意持有的标的，IV 持续偏高
- **注意**：占用保证金较多，$8,500 账户最多同时跑 1-2 个 Wheel

## 二、信用价差策略（占比 20%）

### Bull Put Spread
- **逻辑**：卖高行权价 Put + 买低行权价 Put，看涨/中性
- **最大盈利**：净权利金
- **最大亏损**：价差宽度 - 净权利金
- **理想条件**：IV Rank > 30，有明确支撑位，看涨偏中性
- **推荐价差宽度**：$2-$5（控制最大亏损 ≤ $425）

### Bear Call Spread
- **逻辑**：卖低行权价 Call + 买高行权价 Call，看跌/中性
- **最大盈利**：净权利金
- **最大亏损**：价差宽度 - 净权利金
- **理想条件**：IV Rank > 30，有明确压力位，看跌偏中性

### Iron Condor
- **逻辑**：Bull Put Spread + Bear Call Spread 组合，双向收权利金
- **理想条件**：IV Rank > 50，标的震荡，预期 EM 区间内运行
- **关键指标**：Expected Move — 卖出的行权价要超出 EM 范围
- **止损**：任一方向被突破达到价差宽度 50% 时考虑调整

### Iron Butterfly
- **逻辑**：ATM 卖出 Call + Put，两侧买入保护
- **理想条件**：IV 极高，预期短期回落，标的无明显方向
- **注意**：比 Iron Condor 更激进，胜率更低但权利金更高

## 三、波动率策略（占比 10%，事件驱动）

### Long Straddle
- **逻辑**：同行权价买 Call + Put，押注大幅波动
- **理想条件**：IV 相对历史偏低，即将有催化剂（财报/FDA/宏观）
- **最大亏损**：两腿权利金之和（有限风险）
- **盈利条件**：标的涨跌幅 > 组合成本 / 股价
- **警告**：财报后 IV Crush 可能导致即使标的有方向也亏损

### Long Strangle
- **逻辑**：OTM Call + OTM Put，成本比 Straddle 低
- **理想条件**：同 Straddle，但需要更大的波动才能盈利

### Calendar Spread
- **逻辑**：买远期合约 + 卖近期合约，收 Theta 差
- **理想条件**：近期 IV > 远期 IV，预期标的短期稳定
- **风险**：IV 结构突变，近期 IV 大涨

### Diagonal Spread
- **逻辑**：Calendar Spread 的变种，不同行权价
- **适用**：替代 Covered Call 的合成仓位

## 四、方向性策略（事件驱动时使用）

### Debit Call Spread
- **逻辑**：买低行权价 Call + 卖高行权价 Call，看涨
- **最大亏损**：净权利金（有限风险）
- **理想条件**：IV 偏低适合买方，明确看涨催化剂

### Debit Put Spread
- **逻辑**：买高行权价 Put + 卖低行权价 Put，看跌
- **最大亏损**：净权利金（有限风险）
- **理想条件**：IV 偏低，明确看跌催化剂

## Delta 行权价选择指引

> 用 Delta 定位行权价，比"距当前价 X%"更准确——Delta 已内含 IV 信息。

### 目标 Delta 区间（按策略类型）

| 策略 | 短腿目标 Delta | 对应大致 OTM 程度 | 胜率估算（PoP） |
|------|-------------|----------------|--------------|
| CSP（保守） | Δ0.15–0.20 | 1–1.5 个标准差外 | ~80–85% |
| CSP（标准） | Δ0.20–0.25 | 约 1 个标准差外 | ~75–80% |
| Bull Put Spread 短腿 | Δ0.20–0.25 | 同上 | ~75–80% |
| Bear Call Spread 短腿 | Δ0.20–0.25 | 同上 | ~75–80% |
| Iron Condor 短腿（两侧） | Δ0.15–0.20 | 更保守，减少双边风险 | ~70–75% |
| CC（激进） | Δ0.30–0.40 | 略 OTM，权利金更多 | ~60–70% |

> Delta 估算：ATM ≈ Δ0.50 | 1σ 外 ≈ Δ0.16 | 2σ 外 ≈ Δ0.05
> 精确 Delta 从经纪商平台期权链读取，以上仅为快速定位参考。

### 信用价差宽度选择

| 账户规模 / 单笔风险上限 | 推荐价差宽度 | 最大亏损控制 |
|---------------------|-----------|------------|
| $8,500 账户 / 单笔上限 $425 | $2–$5 | 1–2 张合约时最大亏损 ≤ $425 |
| 权利金收入目标 | 价差宽度 × 25–33% | 例：$5 价差收 $1.25–$1.65 权利金 |

**执行流程**：
1. 确认 IV Rank > 30（信用价差）或 > 40（CSP/CC）
2. 查期权链，找 Delta ≈ 0.20–0.25 的行权价作为短腿
3. 买同方向低 $2–$5 行权价作为保护长腿
4. 确认净权利金 ÷ 价差宽度 ≥ 25%（否则 R/R 不够）
5. 确认最大亏损 ≤ $425（账户 5%）

## 策略选择决策树

```
有无方向判断？
├─ 无明确方向（中性）→ IV Rank 高？
│   ├─ 高（>50）→ Iron Condor / Iron Butterfly
│   ├─ 中（30-50）→ Bull Put Spread / Bear Call Spread
│   └─ 低（<30）→ 暂不操作，等待机会
│
├─ 有方向，IV 偏高 → 卖方策略（CSP / CC / Credit Spread）
│
└─ 有方向，IV 偏低 + 催化剂 → 买方策略（Debit Spread / Straddle）
```
