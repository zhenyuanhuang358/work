核心分析 Core Analysis

技术护城河Technology Moat Cerebras WSE-3 将全部 compute 与 SRAM 集成在单一晶圆上，消除了 GPU 集群中 HBM 与 compute 之间的数据搬运瓶颈。在 LLM token 生成（decode）场景， Figma 实测 Cerebras 约 1,500 t/s，Groq 约为其三分之一，GPU-based 方案如 Baseten 约 250 t/s。 Perplexity Sonar 从 Hopper 的 45–50 t/s 跃升至 Cerebras 的 1,200 t/s。速度优势真实且可验证。

商业化现状Commercialization Status Cerebras 正从硬件销售（单 rack 售价逾 $1M）转向 Cloud inference-as-a-service 模式。 当前已验证的大客户包括 Figma（30% 自研模型流量）、Perplexity（Sonar）、Mistral（Le Chat）。 OpenAI coding model 据前高管表述也运行在 Cerebras 硬件上，但未获官方确认。 最大风险是 G42 约占 2024 年营收 85%，客户结构高度集中。

市场定位Market Positioning Cerebras 的 TAM 取决于口径设定：严格口径（超低延迟刚需场景）当前约 5–10% 总推理流量， 中等口径（reasoning/coding/长回答）约 15–25%。即使在中等口径内，Cerebras 实际可拿份额 受容量、部署形态和软件生态约束，Base Case 预测 2030 年 3% 全球推理份额 → $15B 年收入。 NVIDIA Rubin 正在把低延迟推理纳入全栈，这是对 Cerebras niche 最大的长期挑战。

投资框架Investment Framework 估值驱动因素：(1) 市场份额成立性（1%–5% range 对应 $9B–$75B PV）， (2) G42 依赖是否分散化，(3) TSMC 供应链地缘政治风险，(4) wafer 良率透明度。 催化剂包括 IPO 正式定价、新大客户公告、ML Perf 提交和 MI350/Rubin 上市时间线。 当前适合风险承受能力较强的投资者，建议等待 IPO 定价后评估安全边际。
