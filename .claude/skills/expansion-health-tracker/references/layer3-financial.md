# 第三层：财务传导层操作指南（同步验证指标）

核心问题：**扩张失序已经传导到财务数字了吗？**

数据来源：公司年报 / 半年报 / 季报（上市公司）；非上市公司用行业基准替代。

---

## 信号一：区域收入 / 门店剪刀差

**拆解维度**：分区域统计门店净增速度 vs 该区域收入增速，识别"开店速度快于收入"的区域。

### 数据采集

```
上市公司年报操作：
1. 查找"分部信息"或"分地区收入"披露章节
   - 海底捞：中国大陆按一二线/三线/其他披露
   - 港股年报：通常在"分部信息"附注
2. 提取各区域：收入（近3年）+ 门店数量（近3年）
3. 计算：
   - 区域收入增速（YoY%）
   - 区域门店净增速（YoY%）
   - 剪刀差 = 门店净增速 - 收入增速

WebSearch 补充：
[品牌名] 年报 分部收入 区域 [年份]
[品牌名] annual report segment revenue [年份]
[品牌名] 三线城市 收入 门店数

非上市公司处理：
- 无分部数据 → 标注【数据缺口】
- 替代方案：用窄门餐眼城市层级分布 × 客单价估算区域收入权重
- 标注"【分析预测·估算值】"
```

### 计算方法

```
剪刀差 = 区域门店净增速（%YoY）- 区域收入增速（%YoY）

正常情况：剪刀差 ≈ 0（新店贡献同步拉动收入）
危险信号：剪刀差 > 0 且持续扩大（开店多但收入没跟上）

AUV 稀释验证：
AUV（当期）= 区域总收入 / 区域门店数 / 12
AUV 变化趋势（应与剪刀差方向一致）
```

### 判断阈值

| 状态 | 判断标准 |
|------|---------|
| ✅ 健康 | 剪刀差 < 5ppt；AUV 稳定或上升 |
| ⚠️ 警示 | 剪刀差 5–15ppt；AUV 下滑但幅度 < 10% |
| 🔴 危险 | 剪刀差 > 15ppt；**AUV 系统性下滑 > 10%；三线区域剪刀差明显高于一线** |

### 你的判断介入点

> 区域剪刀差比整体数字更能定位问题：一线收入健康但三线剪刀差扩大，说明扩张失序在三线已经显现。
> AUV 下滑比门店数增加晚 1–2 个季度出现，先看剪刀差再等 AUV 验证。
> 上市公司通常不拆分三线 AUV，需要自己用门店数反算。

## 信号二：翻台率区域截面

**拆解维度**：翻台率的区域分化情况——一线 vs 三线是否已经出现裂口？

### 数据采集

```
上市公司年报操作：
1. 查找"翻台率"或"翻座率"披露（通常在运营数据章节）
2. 确认口径：全天 or 分午市/晚市；直营 or 加盟+直营混合
3. 部分品牌按城市层级披露翻台率（如海底捞），提取各层级数据
4. 若只有整体翻台率：记录趋势 + 与同期门店数对比

WebSearch：
[品牌名] 翻台率 [城市层级] [年份]
[品牌名] table turnover rate [年份] annual report
[品牌名] 一线城市 二线城市 翻台率

历史翻台率基准（参见 baseline-library.md）：
- 直营火锅：健康 3.5–4.5 次/天；危险 < 3.0 次/天
- 非上市公司：用行业基准替代，标注【行业估算】
```

### 计算方法

```
区域翻台率分化指数 = 一线翻台率 - 三线翻台率

正常分化：0.3–0.8 次/天（一线客流密度自然高于三线）
危险信号：分化 > 1.0 次/天，且三线绝对值 < 3.0 次/天

整体翻台率趋势：
同比下滑 > 0.3 次/天 = 客流压力信号
连续 2 个报告期下滑 = 趋势确认
（注：多数餐企仅在年报/半年报披露翻台率，"连续 2 季度"实为"连续 2 个半年报/年报"）
```

### 判断阈值

| 状态 | 整体翻台率趋势 | 区域分化 |
|------|-------------|---------|
| ✅ 健康 | 稳定或同比 < 0.2 次/天下滑 | 分化 < 0.5 次/天 |
| ⚠️ 警示 | 同比下滑 0.2–0.5 次/天 | 分化 0.5–1.0 次/天 |
| 🔴 危险 | 同比下滑 > 0.5 次/天；**或三线绝对值 < 3.0 次/天** | 分化 > 1.0 次/天 |

### 你的判断介入点

> 翻台率是 T-0 指标（同步），但区域分化是领先信号：三线先跌，全国数据还撑着。
> 管理层一般只报整体翻台率。区域分化要靠反推：总收入 ÷ 总门店 ÷ 12 = AUV，再与各区域报告数字对比。
> 翻台率低但营收持平 = 可能靠客单价撑，是需求压力转移，迟早也会暴露。

## 信号三：单店开设成本趋势

**拆解维度**：资本开支（Capex）÷ 新开门店数，识别单店成本是否在上升。

### 数据采集

```
上市公司年报操作：
1. 现金流量表：找"购置物业、厂房及设备"或"资本开支" = 总 Capex
2. 新开门店数（当年净增 + 关闭数）= 新开门店数
3. 年报经营数据章节：部分品牌直接披露"每新店平均投资额"
4. 关闭门店数经常不主动披露，需倒推：
   关闭 = 期初门店数 + 本期新开 - 期末门店数

WebSearch：
[品牌名] 资本开支 capex 新开门店 [年份]
[品牌名] 每店投资额 装修成本 [年份]
[品牌名] 新开门店成本 加盟商 首年回收
```

### 计算方法

```
单店开设成本 = 年度新店相关 Capex / 当年新开门店数

注意：总 Capex 包含维护性 Capex，需拆分：
- 维护性 Capex 估算（误差约 ±20%，仅当公司未单独披露时使用）：
  - 轻资产餐饮（租赁为主、少量固定设备）：维护性 Capex ≈ 折旧 × 0.5–0.7
  - 重资产餐饮（自有设备较多、中央厨房）：维护性 Capex ≈ 折旧 × 0.7–1.0
- 新店 Capex = 总 Capex - 维护性 Capex

如公司直接披露"单店投资额"，直接使用，标注来源。

直营品牌无加盟商数据时，首年回收期替代计算：
首年回收期（估算）= 单店 Capex ÷ 单店年均餐厅层面利润
= 单店 Capex ÷ (AUV × 12 × 餐厅层面利润率)
标注【分析推算·估算值】

成本趋势：计算近 3 年均值及 YoY 变化
```

### 判断阈值

| 状态 | 单店成本趋势 | 含义 |
|------|-----------|------|
| ✅ 健康 | 单店成本稳定，YoY < +15% | 标准化良好，扩张可控 |
| ⚠️ 警示 | 单店成本 YoY +15–30% | 通货膨胀 or 选址成本上升 or 装修升级 |
| 🔴 危险 | 单店成本 YoY > +30%；**或首年回收期 > 24 个月（参见 baseline-library.md）** | 新店投资回报恶化，ROI 压力积累 |

### 你的判断介入点

> 单店成本上升通常滞后：前期开便宜地段，资金不足后才啃贵地段。
> 和翻台率一起看：成本上升 + 翻台率下滑 = ROI 双向挤压，是最危险的组合。
> 加盟模式下，公司 Capex 不含加盟商投入——首年回收期才是真实信号，需要从加盟商社区或媒体报道获取。
