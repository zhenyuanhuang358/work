**止损原则**：
- 卖方策略（CSP/CC/Credit Spread）：损失达权利金收入的 **2倍** 时平仓
- 买方策略（Debit Spread/Straddle）：损失达权利金支出的 **50%** 时评估
- 任何仓位：标的触发重大事件（突发财报/监管）时立即重新评估

## 使用说明

**如何维护**：
1. 每次开仓后，告诉 agent「记录开仓：[标的/策略/参数]」
2. 每周问一次「查看当前敞口」，agent 会汇总风险状态
3. 平仓后告诉 agent「记录平仓：[标的/盈亏/原因]」

**注意**：
- 本日志存于对话上下文，对话结束后不自动保存
- 建议用户将 agent 输出的日志粘贴到本地 Markdown 文件长期维护
- 或告诉 agent「把当前持仓日志写入文件」，agent 会生成可保存的格式
