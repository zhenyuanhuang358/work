核心分析
Core Analysis · Four Dimensions
营收质量 Revenue Quality

营业额¥18.32亿（+6.68%），增长真实但结构分化：香港门店录得+35%高增速，主因2024年基数低叠加旅游复苏；内地仅+4.47%，低于名义消费增速，意味着内地实质上处于存量博弈。对外产品销售（即速食拉面等To-B/C渠道）同比下滑-28.63%，第二曲线持续萎缩，凸显品牌在餐饮之外几乎无定价权。

利润结构 Profit Architecture

毛利率76.98%看似亮眼，但净利率仅1.76%——高达75%的毛利被房租、人工、营销及外卖平台费用几近耗尽。更具警示意义的是H2 vs H1结构：上半年净利润¥3700万，下半年净亏损约¥900万。全年"扭亏"的成绩依赖淡季（H1）支撑，旺季（H2）反而拖累——这与常规餐饮消费规律相悖，值得深度追踪。

成本压力与预期差 Cost Inflation Gap

外卖平台配送费2025H1同比+25.4%（至¥3560万），营销费用+38.2%（至¥1050万）。收入增速6.7%远低于核心成本膨胀率，意味着每新增一元营收，成本扩张已超过利润留存空间。若2026年外卖渗透率继续提升，这一剪刀差将进一步扩大，净利率面临回落至负值的真实风险。

战略执行力 Strategic Execution

管理层于2010年前后喊出「千店目标」，15年后门店数617家，距1000家仍差38%。2025年净增21家，以此速度需再等18年。翻台率自2022年起停止披露（末次已知：内地2.5次/天），从信息披露角度判断，数据大概率未有实质改善。竞争层面，遇见小面以相近营收规模实现味千净利润4倍，差距不在赛道而在运营效率与品牌溢价。

Earner Research Verdict · Executive Summary
执行摘要：3条关键判断
1
「扭亏」是技术性的，不是结构性的。 全年净利润¥2800万，但H1贡献¥3700万、H2亏损约¥900万——意味着全年盈利完全依靠上半年，而通常被视为旺季的下半年反而更弱。这一"反季节盈利"结构无法支撑持续复苏叙事。 依据：2025年报（¥18.32亿收入，¥2800万净利）+ 2025中期报告（H1净利¥3700万）
2
成本通胀正在蚕食毛利优势。 76.98%的毛利率在餐饮业属于高位，但外卖配送费（+25.4%）和营销费用（+38.2%）的增速是营收增速（+6.7%）的4-6倍。若此趋势延续，净利率将在2026年重回负值。高毛利的外表掩盖了真实的盈利脆弱性。 依据：2025中期报告成本拆分数据
3
现价隐含「基准情景」，安全边际极低。 HK$1.06对应P/S约0.5x（基准估值），上行空间受限于净利率天花板，下行风险在H2亏损证实后可能引发估值重估至P/S 0.27x（HK$0.55区间）。当前股价已反映复苏预期，但复苏的可持续性尚未被财务数据证实。 依据：P/S估值模型 + H2 2025盈利结构分析

假设你是分析师：5个核心追问
Key Analyst Questions · If There Were an Earnings Call
味千中国无投资者财报电话。以下为若有电话会议，分析师最可能追问的核心张力点，以及基于年报数据的推断答案。
Q1：全年盈利¥2800万，但下半年亏损约¥900万——管理层如何解释H2的盈利逆转？
年报对H1/H2拆分未作专项说明。可能原因：H2食材、能源成本季节性走高 + 外卖费用集中爆发 + 部分新开店试营业亏损。但结构性原因（品牌号召力下降→H2客流恢复慢于预期）同样不可排除。
张力：管理层"全年扭亏"的表述在技术上正确，但掩盖了H2运营质量的真实状态。
Q2：为何自2022年起不再披露翻台率？
公司未作官方解释。翻台率在2022年已跌至2.5次/天（2010年为5.1次），停止披露通常发生在核心KPI持续走弱的情况下。
张力：停止披露本身即是负面信号——如数据改善，管理层有动力重新发布。
Q3：外卖平台配送费同比+25.4%，营收仅+5.9%，如何维持利润率？
目前靠毛利率高（77%）和绝对成本控制勉强维持净正值，但边际空间极薄。若2026年外卖渗透率继续从当前水平上升5个百分点，每百万元收入对应的配送成本增量约¥4-6万，净利润将完全被抹平。
张力：外卖是结构性趋势，非周期性成本波动，管理层无力反转。
Q4：千店目标已提出15年，2025年仅617家，是否会正式放弃或修订？
年报维持扩张方向，但未给出具体开店数目标。按2025年净增21家速度，距1000家需18年，管理层公信力受损。
张力：坚持不切实际的目标会持续削弱投资者对管理层判断力的信任。
Q5：遇见小面净利润是味千的4倍，具体差距在哪里？
主要差距在于：品牌复购率 → 客流稳定性 → 对外卖平台议价权（遇见小面自带粉丝流量，减少平台依赖）；以及产品迭代速度（遇见小面持续推新，味千产品线老化）。
张力：拉面赛道竞争已经升维，味千的"先发优势"在运营效率上并未形成护城河。
