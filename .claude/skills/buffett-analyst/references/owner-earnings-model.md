# 所有者盈余模型与内在价值计算

---

## 核心公式

### Owner Earnings（所有者盈余）

```
Owner Earnings = 净利润
              + 折旧摊销（D&A）
              - 维护性资本支出（Maintenance Capex）
              ± 运营资本变化（ΔWorking Capital）
```

**为什么不用净利润**：净利润包含非现金项目（折旧、摊销），且 Capex 不走利润表但消耗真实现金。

**维护性 vs 增长性 Capex 的拆分**：
- 理想来源：公司在财报中主动披露（少数公司会做）
- 替代方法：用 D&A 金额代表维护性 Capex（保守估计）
- 如无法拆分：用总 Capex，这是偏保守的处理，可接受

## DCF 两情景模型

### 折现率（r）选取

| 市场 | 推荐折现率 | 逻辑 |
|------|---------|------|
| 港股大盘股 | 9-10% | 无风险利率（美债）+ 股权风险溢价 |
| 港股中小盘 | 11-12% | 额外流动性风险溢价 |
| 美股大盘股 | 9% | 同上 |
| A股 | 10-11% | 人民币资产 + 监管风险溢价 |

**不调整折现率来「使结果合理」**——这是估值最常见的错误。

---

### 情景一：保守情景（Conservative Case）

```
增长假设：g₁ = min(近3年FCF复合增速 × 60%, 护城河评分对应上限)
阶段1：前5年以 g₁ 增长
阶段2：后5年以 g₁ × 50% 增长
终值：以 g_terminal = 2-3%（GDP增速）永续增长，用 Gordon Growth Model

终值 = OE_year10 × (1 + g_terminal) / (r - g_terminal)
```

### 情景二：基准情景（Base Case）

```
增长假设：g₂ = min(近3年FCF复合增速 × 85%, 护城河评分对应上限)
阶段1：前5年以 g₂ 增长
阶段2：后5年以 g₂ × 60% 增长
终值：同上，g_terminal = 3-4%
```

### 内在价值区间

```
保守 IV = PV(保守情景现金流) + PV(保守终值)
基准 IV = PV(基准情景现金流) + PV(基准终值)

最终内在价值区间 = [保守 IV, 基准 IV]
中值（报告使用）= (保守 + 基准) / 2
```

## 安全边际计算

```
安全边际 = (内在价值中值 - 当前市值) / 内在价值中值 × 100%

买入区间：安全边际 ≥ 30%（等于：当前价 ≤ 内在价值 × 70%）
观察区间：安全边际 15-29%
高估区间：安全边际 < 15%（不买，若已持有评估减仓）
```

## 三个关键检验

### 检验 1：FCF 转化率
```
FCF 转化率 = FCF / 净利润

正常：75-100%（FCF ≈ 净利润）
优秀：> 100%（FCF 持续超过净利润 = 高质量盈利）
警告：< 60%（连续3年）= 会计利润虚高，需检查 Capex 或应收账款
```

### 检验 2：ROIC vs 折现率
```
若 ROIC < r（折现率）：公司投入的每块钱在摧毁价值，增长越快越糟糕
若 ROIC > r × 1.5：高质量复利机器

ROIC = NOPAT / 投入资本
NOPAT = 营业利润 × (1 - 税率)
投入资本 = 股东权益 + 有息负债 - 现金
```

### 检验 3：资本分配质量
| 管理层行为 | 分数 |
|-----------|------|
| 当股价低于内在价值时回购 | +2 |
| 分红且连续增长 | +1 |
| 高价并购（溢价 > 40% 且无明确协同效应） | -2 |
| 大规模股权激励稀释 | -1 |
| 债务扩张超过 FCF 增速 | -2 |

## 估值失效条件（放弃 DCF，转定性评估）

以下情况 DCF 数字不可信，必须在报告中声明：

1. FCF 连续3年为负（尚在投入期，终值主导估值 = 不可靠）
2. 营收增速波动率 > 50%（每年大幅变化，增长率无法合理外推）
3. 负债率 > 50% 且 EBIT/利息 < 3x（财务风险主导，非运营质量）
4. 行业处于根本性技术颠覆期（5年现金流不可预期）

这种情况下：改用 EV/FCF 与同类可比公司区间估算，但必须注明「定性估值，可信度较低」。
