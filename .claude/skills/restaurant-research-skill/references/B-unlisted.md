# B类：非上市规模连锁数据采集策略

适用：门店>100家但未上市的餐饮连锁。财务数据通常不可得，重点从媒体、招股书（如有）、企查查构建拼图。

---

## 采集优先级与搜索策略

### 1. 媒体深度报道
- 搜索：`[品牌名] 门店数 营收 扩张 [年份]`
- 搜索：`[品牌名] 融资 估值 IPO 上市计划`
- 来源：晚点、36氪、虎嗅、餐饮O2O、红餐网
- **窄播 Jelly（对非上市连锁尤佳）**：`[品牌名] site:jelly.tech` 或搜`[品牌名] 窄播`

**数据来源冲突处理**：若晚点与36氪给出不同门店数/营收 → 取较新来源，并列注明差异；差距>20%时标注`【来源有分歧，建议IR核实】`。

### 2. 招股书 / 港交所上市申请（仅适用于有IPO进程的品牌）

> ⚠️ 绝大多数 B 类非上市连锁无 IPO 计划，此路径适用性有限，按实际情况判断是否搜索。

- 港交所 A1 上市申请：`[品牌名] listing application site:hkexnews.hk`（A1 申请含3年完整历史财务数据）
- A股招股说明书：`[品牌名] 招股说明书 site:cninfo.com.cn`
- 价值：是非上市公司财务数据最完整的来源，优先于媒体报道数字

### 3. 企查查/天眼查
- 搜索：`[品牌名] 企查查 工商信息 股权结构 投资人`
- 可获取：注册资本、股东结构、历次融资记录、关联公司

### 4. 行业数据交叉验证
- 搜索：`[细分品类] 连锁规模排名 市场份额 [年份]`
- 来源：中国连锁餐饮协会、美团研究院公开报告、艾瑞咨询

### 5. 可比公司估值推算（估算市值时使用）

若需估算非上市品牌的价值：
1. 查同赛道上市公司当前 **P/S（市销率）** 倍数
2. 乘以媒体引述的估算营收
3. 示例：同赛道上市公司 P/S = 3–5×，品牌营收估算 ¥10 亿 → 估值 ¥30–50 亿
4. 标注`【分析预测·可比估算】`，注明参考公司名称和估值日期

---

## 关键数据采集清单

```
□ 门店总数及近期增速（媒体/企查查来源，注明）
□ 融资历史：轮次、金额、投资方、时间
□ 最新估值（融资公告或媒体引述）
□ 营收体量（通常只有区间或媒体引述，标注为【二手·媒体】）
□ 直营/加盟模型及费用结构
□ 扩张目标（管理层公开表述）
□ 近期重要战略动作（新品/新市场/供应链调整）
□ 创始人/管理层背景
□ 竞品对比位置
```

---

## 特别说明

非上市公司财务数据几乎不可得，在报告中统一声明：

> "该品牌为非上市公司，无公开财务数据。以下数据基于媒体报道和公开渠道，标注`【二手·媒体】`；财务分析依据行业可比公司估算，标注`【分析预测】`。"

不要用模糊表述掩盖这一局限——客户需要知道数据质量。
