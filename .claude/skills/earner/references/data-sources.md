# 数据采集策略（无 Transcript 时）

当用户未提供财报电话记录时，按以下策略自主采集。  
**采集完成前不启动分析。**

---

## 市场类型判断

| Ticker 特征 | 市场 | 采集策略 |
|-------------|------|---------|
| 纯数字 + `.HK`（如 `00538.HK`）| 港股 | 策略 B1 |
| 6位数字 + `.SS`/`.SZ`（如 `600519.SS`）| A股 | 策略 B2 |
| 纯字母 2–5位（如 `AMD`、`NVDA`）| 美股 | 先搜 transcript；失败则策略 B3 |
| 其他 | 未知 | WebSearch 先确认市场，再选策略 |

---

## 策略 B1：港股

**报告周期**：半年报（8–9月）+ 年报（3–4月）

**搜索顺序（按优先级）：**

```
1. "[公司名] [年份] 年报 OR 中期报告 site:hkexnews.hk"
2. "[公司名] [年份] 业绩 营收 净利润 site:aastocks.com"
3. "[公司名] [股票代码] [年份] 年度业绩 扭亏 营收"
4. "[公司名] [股票代码] analyst forecast consensus [年份]"
```

**必须提取（有则取，无则标 `badge-gap`）：**
- 营业额（人民币百万元）及 YoY%
- 归母净利润及 YoY%
- 毛利率
- 门店数（餐饮必选）/ 期末经营指标
- 管理层报告中的展望原文（直接引用，不加工）
- 股本（总股数，用于 EPS 计算）

**货币换算规则（港股）：**
- 港股公司通常以人民币报告内地业务，以港元报告集团合并报表
- 报告中保留原始货币，目标价统一以 HKD 表示
- RMB → HKD：乘以 1.10（约估，注明换算率）
- HKD → USD：除以 7.78（约估，注明换算率）
- **绝对禁止**：在同一数字比较中混用 RMB 和 HKD，不注明单位

## 策略 B2：A股

**报告周期**：季报（4/8/10月）+ 半年报（8月）+ 年报（4月）

**搜索顺序：**

```
1. "[公司名] [股票代码] [年份] 年报 OR 季报 site:cninfo.com.cn"
2. "[公司名] [年份] 营业收入 净利润 财报"
3. "[公司名] [年份] 业绩快报 业绩预告"
4. "[公司名] 研报 目标价 [年份] site:eastmoney.com"
```

**货币**：人民币（元），目标价以 RMB 表示。

---

## 策略 B3：美股（无 Transcript）

**搜索顺序：**

```
1. "[TICKER] Q[N] [年份] earnings call transcript site:seekingalpha.com OR site:fool.com"
2. "[TICKER] [季度] [年份] earnings revenue EPS beat miss"
3. "[TICKER] [年份] annual report 10-K site:sec.gov"
4. "[TICKER] consensus estimate EPS revenue [年份]"
```

若找到 transcript 全文 → 切换到路径 A（transcript 解析）。

---

## 共同规则

**数据质量分级**：

| 来源 | Badge | 可信度 |
|------|-------|--------|
| 官方年报 / 10-K / 港交所公告 | `badge-official` | 高 |
| 业绩快报 / 盈利预警 | `badge-official` | 高 |
| 彭博/路透共识预期 / 卖方研报数字 | `badge-predict` | 中 |
| 新闻报道引用的数字 | `badge-predict` | 中低 |
| 无法找到 | `badge-gap` | — |

**停止披露的 KPI**：
- 若历史报告曾披露某指标（如翻台率），近期停止 → 自动标注 `badge-gap` 并在报告中说明「[年份]后停止披露，可能反映指标持续走弱」

**数据采集完成检查**：
```
□ 最近 2–3 个报告期的营收已获取（至少 1 期）？
□ 净利润 / 归母净利润已获取？
□ 毛利率已获取或可推算？
□ 股本数量已获取（用于 EPS）？
□ 管理层展望原文已引用（哪怕只有一句）？
□ 货币单位已确认且一致？
```
