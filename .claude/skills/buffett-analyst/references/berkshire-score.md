# 巴菲特评分卡（Berkshire Score）

满分 100 分。每个维度的分数基于可查数据，不允许主观感觉填分。

---

## 维度一：Business Quality（30分）

### 1A. 护城河强度（0-15分）

| 护城河评分（moat-framework.md） | 得分 |
|-------------------------------|------|
| 0-2（无护城河） | 0（终止分析） |
| 3-4（窄护城河） | 5 |
| 5-6（中等） | 8 |
| 7-8（宽护城河） | 12 |
| 9-10（卓越护城河） | 15 |

### 1B. 收入稳定性（0-8分）

| 条件 | 得分 |
|------|------|
| 近3年营收 CAGR > 20% 且无负增长年份 | 8 |
| 近3年营收 CAGR 10-20%，最多1年负增长 | 6 |
| 近3年营收 CAGR 5-10%，稳定 | 4 |
| 增速不稳定（波动率高）或增速 < 5% | 2 |
| 多年营收下滑 | 0 |

### 1C. 定价权证据（0-7分）

| 条件 | 得分 |
|------|------|
| 毛利率近3年稳定 > 50% | 7 |
| 毛利率近3年稳定 35-50% | 5 |
| 毛利率 20-35% 且无明显压缩 | 3 |
| 毛利率持续压缩或 < 20% | 1 |

## 维度二：Financial Health（30分）

### 2A. 资本回报率（0-12分）

| 条件 | 得分 |
|------|------|
| ROE 3年均值 > 25% 且无大量杠杆驱动 | 12 |
| ROE 3年均值 18-25% | 9 |
| ROE 3年均值 12-18% | 6 |
| ROE 3年均值 8-12% | 3 |
| ROE < 8% 或主要靠债务杠杆 | 0 |

> **杠杆检验**：若负债率 > 40%，ROE 需用 ROIC 验证。ROIC < 12% 则降2档。

### 2B. FCF 质量（0-10分）

| 条件 | 得分 |
|------|------|
| FCF 连续3年为正 + FCF/净利润 > 80% | 10 |
| FCF 连续3年为正 + FCF/净利润 60-80% | 7 |
| FCF 2年正1年负，整体趋势向好 | 5 |
| FCF 不稳定或持续低于净利润 50% | 2 |
| FCF 多年为负 | 0 |

### 2C. 资产负债表健康（0-8分）

| 条件 | 得分 |
|------|------|
| 净现金（现金 > 总有息负债） | 8 |
| 负债率 < 20% 且 EBIT/利息 > 10x | 6 |
| 负债率 20-40% 且 EBIT/利息 > 5x | 4 |
| 负债率 40-60% 或 EBIT/利息 3-5x | 2 |
| 负债率 > 60% 或 EBIT/利息 < 3x | 0 |

## 维度三：Management Quality（25分）

### 3A. 资本分配记录（0-12分）

基于近3年实际决策（不是管理层说的，是做的）。从下表中逐项判断，加总后截断在 [0, 12]：

**基础分：6分**（代表"没有明显失误，也没有明显亮点"）

| 行为（有证据才加/减） | 调整 |
|---------------------|------|
| 在低估值时回购（当时 PB < 2 或 PE < 15，有公告可查） | +4 |
| 分红连续增长 3 年以上 | +2 |
| 投资有明确 ROIC > 20% 的内部项目（有数据支撑） | +2 |
| 高溢价并购后商誉减值（有减值公告） | -4 |
| 股权激励连续 3 年稀释 > 3%/年 | -3 |
| 举债并购且 EBIT/利息 < 3x（可查财报） | -4 |

最终得分 = max(0, min(12, 6 + 加减合计))

### 3B. 管理层与股东利益一致性（0-8分）

| 条件 | 得分 |
|------|------|
| 创始人仍在主导 + 持股比例 > 20% | 8 |
| 创始人主导 + 持股 10-20% | 6 |
| 职业经理人 + 大量持股（薪酬激励结构良好）| 5 |
| 职业经理人 + 少量持股 | 3 |
| 股权分散 + 无明确绑定激励 | 1 |

### 3C. 沟通诚实度（0-5分）

基于近2年财报、电话会议、投资者日：

| 条件 | 得分 |
|------|------|
| 主动披露风险和失误，承认错误 | 5 |
| 中性，没有明显粉饰，也没有深度披露 | 3 |
| 措辞模糊，回避关键指标（如毛利率分部数据） | 1 |
| 有误导性陈述或数据前后矛盾 | 0 |

## 评分结果解读

| 总分 | 结论 | 说明 |
|------|------|------|
| 85-100 | **强力买入区域** | 极罕见；巴菲特式机会，核心仓位 |
| 70-84 | **等待买入** | 质量优异，等安全边际达标后建仓 |
| 55-69 | **观察名单** | 有投资价值，但护城河或估值存在明显缺陷 |
| 40-54 | **谨慎** | 部分优点，但关键指标未达标，风险与收益不匹配 |
| < 40 | **排除** | 不符合巴菲特标准，资源应分配给更好机会 |

## 自动降级规则（覆盖总分，分两档）

### 直接「排除」（无论总分多高，终止分析）
- 近12个月发现财务造假或重大信息披露违规
- 护城河评分 ≤ 2（无任何可辨识护城河，商业模式不成立）

### 强制降至「谨慎」（不得给出买入建议）
- 护城河评分 3（窄护城河边缘，需单独说明不确定性）
- 负债率 > 70% 且 EBIT/利息 < 2x
- FCF 连续3年为负（尚未证明可持续盈利能力）

在报告中用醒目方式注明触发了哪条规则及原因。
