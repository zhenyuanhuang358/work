# 产业链梳理框架

调研提纲含"产业链"/"上下游"/"供应链"/"竞争格局"时加载。

核心问题：**品牌在产业链哪个位置？对上游议价能力强不强？对下游粘性足不足？**

---

## 一、通用分析框架

```
上游（原材料/设备/包装）→ 中游（加工/中央厨房/冷链/配送）→ 品牌（门店端）→ 下游（消费者/平台）
```

每层分析四个维度：

| 维度 | 核心问题 | 对品牌的影响 |
|------|---------|------------|
| **参与者集中度** | 这一层有几家主要玩家？是垄断还是分散？ | 集中 → 对方议价强；分散 → 品牌主导 |
| **利润分配** | 这一层毛利率/净利率是多少？ | 利润集中在哪层，那层就有护城河 |
| **议价能力** | 品牌能否锁定价格/排他合作？ | 影响成本稳定性和供应安全性 |
| **替代难度** | 换供应商/平台的切换成本高不高？ | 低替代难度 = 品牌对供应链控制力弱 |

## 二、品类产业链模板

### 火锅（含涮烤）

```
【上游】
食材端：
- 牛羊肉：主要供应商（内蒙古/西北牧场为主）；集中度中等，季节性强
- 海鲜/毛肚：水产供应商，供应链更分散
- 蔬菜/豆制品：区域化采购，标准化难度高

底料/调味品端：
- 上市供应商：颐海国际（1579.HK）、天味食品（603027.SH）
- 关键判断：品牌是自产底料（如巴奴、海底捞部分）还是外购？
  自产 → 供应链控制力强，毛利更高；外购 → 成本暴露给供应商

冷链/配送：
- 顺丰冷链、安得智联、各地区域冷链服务商
- 关键判断：门店数超过供应链覆盖半径后，配送成本是否骤升？
  （这是 expansion-health-tracker 第二层"口味类差评↑"的根本原因之一）

【中游】
中央厨房：
- 标准化程度 = 品牌扩张可复制性的核心指标
- 覆盖半径（冷链通常 2–4 小时车程）→ 决定可扩张城市圈
- 判断：品牌是否在新城市建立区域中央厨房？还是从总部辐射？

【门店端（品牌层）】
直营 vs 加盟：影响供应链控制力和利润分配
人均：¥50–300（跨度大，影响上游采购规格）

【下游】
到店消费为主；外卖占比通常低（涮锅品类体验损耗大）
美团/饿了么平台议价能力：火锅外卖占比低，平台对品牌的控制力弱于快餐
```

```

[品牌名] 颐海国际 供应 自产 底料
颐海国际 海底捞 关联交易 [年份] 年报
[品牌名] 冷链 配送 覆盖城市 半径
```

### 快餐（汉堡/中式快餐/米粉米线）

```
【上游】
肉类/食材：集中度高（百胜/麦当劳有全国锁定供应商）；中小品牌采购分散
酱料/半成品：标准化程度决定品牌复制能力
包装/容器：成本低但环保合规压力增大

【中游】
中央工厂/委托代工：
- 大品牌通常自建；中等品牌委托代工
- 委托代工 = 配方泄露风险 + 质量控制难度↑

物流：
- 常温/冷鲜运输比例；对比火锅，快餐物流标准化更高

【门店端】
坪效（日均销售额/平方米）是核心密度指标
加盟比例高的品牌：供应链收入（卖给加盟商的食材/包装）= 核心利润来源

【下游】
外卖占比高（通常 30–50%，【行业估算·快餐业】）→ 美团/饿了么抽佣压力大
私域流量（会员/小程序）= 对冲平台议价的关键资产
```

**搜索模板**：
```
[品牌名] 供应链 中央工厂 委托代工 [年份]
[品牌名] 加盟商 食材供应 供应链收入
[品牌名] 外卖占比 美团 平台佣金 [年份]
[品牌名] 会员 私域 小程序 用户量
```

### 茶饮（新茶饮/奶茶）

```
【上游】
茶叶：核心品类，中高端品牌倾向定制原料茶，锁定核心供应商
鲜奶/乳制品：成本敏感，部分品牌锁定有机认证供应商作差异化
水果：季节性强，采购管理能力 = 品类上新速度的护城河
糖/包装：成本低，无显著议价博弈

【中游】
茶叶来源：以区域性中小茶场为主（闽/滇/粤产区），暂无在新茶饮供应链中具主导地位的上市供应商；
部分品牌通过与茶产区政府/农协直采合作锁定货源
乳制品：蒙牛/伊利为主要合作方（上市公司）
无中央厨房（茶饮门店即制即售，中游标准化程度低）

【门店端】
平均客单价：¥15–40，高频低决策
出杯量/天 = 核心运营密度指标（翻台率的替代指标）

【下游】
外卖占比高：40–60%（【行业估算·茶饮业】），平台议价压力最大的品类之一
社交传播（小红书/抖音）= 新品上市 → 排队爆款 → 自然流量，是降低平台依赖的关键
```

**搜索模板**：
```

[品牌名] 上新频率 新品 研发 供应链
[品牌名] 外卖 美团 饿了么 占比

```

## 三、供应链风险早期信号

| 风险类型 | 可观测信号 | 数据来源 |
|---------|---------|---------|
| **供应链覆盖半径不足** | 口味类差评季度占比上升（expansion-tracker Layer 2） | 大众点评差评 |
| **关键供应商集中风险** | 单一供应商收入占比 >30%（上市公司：年报附注关联交易披露；非上市：媒体/招股书） | 年报附注 / 招股书 |
| **原材料成本波动未锁定** | 毛利率与大宗商品价格走势强相关 | 财报 + 商品价格指数 |
| **中央厨房产能瓶颈** | 扩张速率加速但区域收入增速低于门店增速 | expansion-tracker Layer 3 |
| **冷链覆盖断层** | 新开城市（三线）出现集中食安投诉 | 黑猫投诉 + 大众点评 |
