## HTML 输出规范（用户请求「出 HTML」时使用）

风格参照 Earnings Copilot 模板：
- 配色：ink `#0a0a0b` / paper `#f1efea` / copper `#8b6c42` / gold `#c9a84c`
- 字体：IM Fell English（英文）+ Songti SC（中文）
- 护城河评分：用 0-10 滑动条 SVG 可视化
- 价格锚点：用水平 SVG 区间图（保守 IV / 基准 IV / 当前价 / 买入目标价）
- 巴菲特评分卡：四维雷达图或分段进度条
- 财务指标：三年趋势折线图

**零 Markdown 符号**，所有格式通过 HTML/CSS 实现。
