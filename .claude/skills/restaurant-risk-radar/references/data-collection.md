# 数据感知 — 阶段零执行手册

诊断开始前必须完成，目的是用客观数据预装信号，减少对老板主观描述的依赖。

---

## 步骤 0a：公开数据采集

询问品牌名称后，并行搜索以下五类数据源。每类搜索完成后立即标注：该数据对应哪个失控原型的哪个早期信号。

---

### 数据源 1：门店规模与扩张速度

**搜索词**：`[品牌名] 门店数量 扩张 2025 2026`、`[品牌名] 新店开业`

**提取信息**：
- 当前总门店数
- 过去12个月新增门店数（扩张速度）
- 是否有明确的扩张计划公告

**信号映射**：
| 发现 | 对应原型 |
|------|---------|
| 12个月内门店数翻倍以上 | 扩张稀释 / 节奏错位 |
| 扩张速度加速，无减速信号 | 节奏错位 |
| 媒体报道新店频繁出问题 | 节奏错位 |

---

### 数据源 2：消费者评分分布

**搜索词**：`[品牌名] 大众点评 评分`、`[品牌名] 美团 评价 投诉`

**提取信息**：
- 品牌整体评分（vs 同品类均值）
- 不同城市/门店评分差异（有没有「同一品牌，口碑两极」的现象）
- 高频投诉关键词（「跟以前不一样」「看店看命」「这家比那家差远了」）

**信号映射**：
| 发现 | 对应原型 |
|------|---------|
| 各门店评分差异 > 0.5分 | 标准漂移 |
| 投诉集中在「不一致」「看运气」 | 标准漂移 |
| 整体评分连续下滑 | 标准漂移 / 文化断层 |
| 老店评分明显高于新店 | 扩张稀释 / 节奏错位 |

---

### 数据源 3：招聘数据

**搜索词**：`[品牌名] 招聘 店长`、`[品牌名] 区域经理 岗位`（在 Boss直聘 / 拉勾 / 智联招聘 搜索）

**提取信息**：
- 当前在招店长数量（与门店总数的比例）
- 区域经理在招数量
- 岗位描述中的经验要求（「3年以上」说明内部无人可提拔）
- 薪资范围（偏低说明人才竞争力弱）

**信号映射**：
| 发现 | 对应原型 |
|------|---------|
| 在招店长数量 > 门店总数 5% | 造血失能 |
| 岗位要求「有同类品牌经验」 | 造血失能（内部培养失效） |
| 区域经理招聘量大 | 管理半径超载（现有人撑不住） |
| 门店扩张快但招聘店长更快 | 造血失能 + 节奏错位 |

---

### 数据源 4：企业违规记录

**搜索词**：`[品牌名] 企查查`、`[品牌名] 天眼查 行政处罚`、`[品牌名] 劳动仲裁`

**提取信息**：
- 近2年行政处罚记录（食品安全 / 劳动纠纷 / 税务）
- 劳动仲裁案件数量（员工离职纠纷多 = 流失率信号）
- 法律诉讼（供应商纠纷 = 现金流压力信号）

**信号映射**：
| 发现 | 对应原型 |
|------|---------|
| 食品安全处罚 > 2次/年 | 标准漂移 |
| 劳动纠纷案件多 | 造血失能 / 文化断层 |
| 供应商付款纠纷 | 利润幻觉（现金流紧张） |
| 多家门店分别被处罚 | 管理半径超载（巡店形同虚设） |

---

### 数据源 5：近期新闻与舆情

**搜索词**：`[品牌名] 2026`、`[品牌名] 员工 投诉 离职`、`[品牌名] 加盟 问题`

**提取信息**：
- 媒体正面/负面报道比例
- 员工匿名投诉内容（脉脉 / 职场圈）
- 是否有前员工/加盟商公开发声

**信号映射**：
| 发现 | 对应原型 |
|------|---------|
| 员工投诉「管理混乱」「没人管」 | 管理半径超载 |
| 加盟商投诉「总部不管事」 | 节奏错位 |
| 前骨干员工离职后公开评价差 | 文化断层 / 造血失能 |
| 媒体报道「扩张过快」「品控下滑」 | 扩张稀释 |

---

## 步骤 0b：用户提供资料解读

用户发来资料时（文字整理 / 内部数据 / 访谈记录 / 截图描述），按以下顺序处理：

1. **提取硬数据**：数字类信息（门店数、翻台率、流失率、利润率、区域经理管店数）
2. **提取软信号**：描述类信息（员工怎么说、区域经理反馈了什么、老板自己感受到什么）
3. **与公开数据交叉验证**：
   - 一致 → 信号强度加倍，置信度高
   - 矛盾 → 标注出来，在阶段一追问澄清
   - 用户资料独有 → 标注「内部信号」，优先级高于公开数据

**用户资料优先级高于公开数据**，因为老板掌握外部看不到的真实运营信息。

---

## 数据快照输出格式

阶段零完成后，输出以下格式（简洁版，不超过200字）：

```
📊 数据快照 — [品牌名]

规模：[X] 家门店，过去12个月新增 [X] 家
评分：大众点评均分 [X]，跨店差异 [描述]
招聘：当前在招店长 [X] 个，区域经理 [X] 个
违规：近2年行政处罚 [X] 次
舆情：[一句话总结主要风险信号]

⚠️ 预发现信号：
· [信号1]（来源：[数据源]）→ 疑似 [原型名称]
· [信号2]（来源：[数据源]）→ 疑似 [原型名称]

以下问题还需你来回答：[列出数据盲区，进入阶段一]
```
