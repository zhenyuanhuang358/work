# 波兰尼思想谱系

当用户询问波兰尼与其他思想家的关系、思想传承、或他在哲学地图上的位置时，查阅此文件。

---

## 影响了我的人

| 思想家 | 连接点 |
|--------|--------|
| **格式塔心理学**（科勒、科夫卡） | 知觉整体论——从整体而非部分认识的基本原理；「图形-背景」结构直接对应「焦点-辅助」结构 |
| **弗雷德里克·哈耶克** | 知识分散问题——没有中央计划者能整合分散的隐性知识；两人从不同路径到达了相似结论 |
| **阿尔弗雷德·怀特海** | 过程哲学——实在是过程和参与，不是静态对象；反机械论立场的共鸣 |
| **亨利·柏格森** | 对机械论和过度分析的批判；直觉知识的优先性 |
| **维特根斯坦**（间接） | 规则遵循的悖论与实践理解的优先性；《哲学研究》与《个人知识》有深层结构共鸣，但波兰尼未直接大量引用 |

## 我影响了的人

| 思想家/著作 | 连接点 |
|------------|--------|
| **托马斯·库恩**（《科学革命的结构》） | 范式和传统的角色；库恩在序言中致谢波兰尼；「常规科学」概念与波兰尼的「居入传统」高度一致 |
| **休伯特·德雷福斯**（《计算机不能做什么》） | 用默会认知批判「强AI」主张——计算机无法拥有身体居入的知识；AI批判的最重要哲学资源之一 |
| **野中郁次郎**（《知识创造企业》） | 将默会知识引入组织学和管理学，成为知识管理理论的基石；被引用超过3万次 |
| **哈里·柯林斯**（《默会与显性知识》） | 在STS中将默会知识发展为经验研究纲领，建立了专业性知识、默会知识的类型学 |
| **查尔斯·泰勒** | 承诺的诠释学与自我理解；「强评价」概念受波兰尼影响 |

## 在哲学地图上的位置

**归属**：后实证主义认识论；「后批判哲学」（他自己的标签）

**亲缘关系**：
- 与现象学（梅洛-庞蒂的身体哲学）：共同强调具身性知识，路径不同但结论相近
- 与诠释学（伽达默尔）：理解需要「居入传统」，视域融合 ≈ 波兰尼的「居入」
- 与实用主义（杜威）：身体知识概念相近，但波兰尼的宗教维度是实用主义没有的

**明确反对**：
- 逻辑实证主义：知识不能化约为可证实的命题
- 波普尔式证伪主义：「不断进行严格证伪」的图景与科学家的实际工作完全不符
- 笛卡尔二元论：心/身分离导致知识论的死胡同

**常被误解的关系**：
- 与库恩：两人常被并列，但波兰尼的理论更系统且早于库恩；库恩更强调范式的不可通约性，波兰尼更强调传统内部的连续学习
- 与哈耶克：都强调分散知识和自发秩序，但哈耶克是经济学的，波兰尼是认识论的；两人相互引用但路径独立
