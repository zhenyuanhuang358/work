# C类：新兴品牌数据采集策略

适用：门店<100家或成立<5年的早期餐饮品牌。数据极度稀缺，重点分析模式和差异化，不强求财务数据。

---

## 采集优先级与搜索策略

### 1. 融资信息（最可靠的规模信号）
- 搜索：`[品牌名] 融资 投资 VC 轮次 [年份]`
- 来源：IT桔子、36氪融资快讯、企查查
- 价值：融资金额/轮次往往是门店数/营收规模的间接信号

### 2. 媒体报道（侧重品牌叙事）
- 搜索：`[品牌名] 品牌故事 创始人 模式 扩张`
- 重点：模式创新点、核心差异化、创始人背景

### 3. 社媒信号（消费者端验证）
- 搜索：`[品牌名] 小红书 抖音 口碑 排队 爆款`
- 价值：反映消费者认知、热度趋势、产品力信号
- 限制：社媒热度≠经营质量，不能作为核心结论依据

### 4. 点评平台
- 搜索：`[品牌名] 大众点评 评分 门店分布`
- 可估算：门店数量、主要城市覆盖、用户评价趋势

---

## 关键数据采集清单

```
□ 成立时间、品牌发布时间
□ 创始人/团队背景（有无行业经验）
□ 融资进展（轮次、金额、投资方）
□ 当前门店数量及城市分布
□ 商业模式（客单价定位、直营/加盟比例）
□ 核心产品/差异化卖点
□ 社媒热度信号（小红书/抖音提及量、话题量）
□ 扩张计划（媒体引述）
```

---

## 关键分析视角：趋势 > 绝对值

新兴品牌当前体量通常很小，绝对数字参考价值有限。重点判断**方向和速度**：

| 判断维度 | 正面信号 | 警示信号 | 数据获取方式 |
|---------|---------|---------|------------|
| 品牌热度 | 媒体报道密度月环比上升 | 半年无深度报道 | 搜索近3个月 vs. 去年同期报道量对比 |
| 扩张质量 | 开城等级稳定，首店选核心商圈 | 急速下沉二三线，选址质量下滑 | 大众点评门店列表城市分布 |
| 社媒轨迹 | 抖音主账号粉丝/播放量持续增长 | 热度脉冲式（营销后归零） | 搜`[品牌名] 抖音 粉丝 增长` |
| 融资质量 | 知名消费基金跟投后续轮次 | 单一财务投资人，无消费行业专家 | IT桔子 |
| 口碑稳定 | 大众点评评分 4.5+，评论量持续增长 | 评分下滑或评论数停增 | 点评页面趋势 |

**融资金额与规模的粗略关联（参考区间，非公式）**：

| 融资轮次 / 金额 | 通常对应的门店规模 |
|--------------|----------------|
| 天使 / Pre-A，< ¥3000 万 | 5–30 家，试验期 |
| A 轮，¥3000 万–¥1 亿 | 20–100 家，扩张启动 |
| B 轮，¥1 亿–¥5 亿 | 100–500 家，规模化 |
| C 轮及以上 | 500+ 家或海外扩张 |

> 外卖/虚拟品牌与实体店差异大；融资估算门店数时标注`【分析预测·参考区间】`。

## 特别说明

在章节开头统一声明数据局限（不逐条标注，避免报告视觉过重）：

> "该品牌为早期新兴品牌，公开数据极为有限。以下分析主要基于媒体报道和社媒信号，财务数据均为估算，标注`【分析预测】`。建议客户通过一手渠道（实地走访/行业人脉）补充核心数据。"
