# 传播力评分框架

详细分析框架。用于评估每个话题的爆文潜力，并撰写「为什么会爆」章节。

---

## 话题生命周期判断（每个话题必须先判断，再评分）

| 状态 | 标志 | 建议 |
|------|------|------|
| **↑ 爆发中** | 今日内从零爆发，热度曲线陡升，评论仍在涌入 | 立即行动，窗口 12-24h |
| **⊙ 峰值区** | 已在热榜 1-2 天，话题量高但增速放缓，大 V 开始跟进 | 今日内发，明日红利减半 |
| **↓ 消退中** | 超过 48h 且无新进展，榜单排名下滑 | 除非有新角度，否则跳过 |
| **👀 苗头期** | 刚出现，仅 1 个平台少量讨论，但话题质量高 | 提前布局，等爆发再跟进 |

> 「↓ 消退中」的话题：若有重大新进展（新数据/反转/新当事人发声）可重新评为「↑」

## 传播力综合评分（满分 10 分）

### A. 热度速度（最高 3 分）

| 情况 | 得分 |
|------|------|
| 过去 2 小时内从 0 爆发 | 3 |
| 今天出现，快速上升中 | 2 |
| 今日出现，增速一般 | 1 |
| 昨日或更早，热度平稳 | 0 |

### B. 情绪强度（最高 3 分）

| 情绪类型 | 传播力 | 得分参考 |
|---------|-------|---------|
| 愤怒 | 最强 | 2-3 |
| 焦虑 / 恐惧 | 强 | 2-3 |
| 羡慕 / 无力感 | 强 | 2 |
| 惊讶 / 反常识 | 中强 | 2 |
| 希望 / 励志 | 中 | 1-2 |
| 纯信息 / 无情绪 | 弱 | 0 |

### C. 冲突与对立（最高 2 分）

| 情况 | 得分 |
|------|------|
| 两极化对立，网友撕裂 | 2 |
| 有明确争议点 | 1 |
| 无冲突，单一声音 | 0 |

### D. 跨平台扩散（最高 2 分）

| 情况 | 得分 |
|------|------|
| 3 个以上平台同时热 | 2 |
| 2 个平台出现 | 1 |
| 仅 1 个平台 | 0 |

**总分 ≥ 7 → 强烈推荐；5-6 → 有潜力；< 5 → 跳过**

## 情绪分析模板

分析「为什么会爆」时，逐一拆解：

### 1. 核心情绪
这个话题触发的是哪种原始情绪？（只选 1-2 个主要情绪）

- **愤怒**：有没有明确的「坏人」/「加害者」/「不公平」？
- **焦虑**：是否触碰了某个群体对未来的恐惧（工作/收入/阶层下滑）？
- **羡慕**：是否展示了巨大的财富差距或人生差距？
- **无力感**：是否让人感到「普通人赢不了」？
- **惊讶**：是否颠覆了已有认知？

### 2. 身份共鸣
这个话题让哪个群体感到「说的就是我」？

常见高传播群体：
- 打工人（996/降薪/裁员/职场不公）
- 中产（买房/子女教育/消费降级/财富缩水）
- 年轻人（高房价/低工资/躺平/内卷）
- 创业者（融资/AI冲击/监管/竞争）
- 父母（教育焦虑/医疗/养老）
- 女性（职场歧视/婚育压力/消费）

### 3. 冲突结构
识别话题内部的对立结构：

| 冲突类型 | 举例 |
|---------|------|
| 个人 vs 平台/公司 | 员工被裁/用户数据泄露 |
| 普通人 vs 富人 | 财富差距/税收优惠 |
| 国内 vs 国际 | 产品出海/技术封锁 |
| 传统 vs AI | AI 替代岗位/效率革命 |
| 老一代 vs 新一代 | 职场观念/消费观 |
| 专家/权威 vs 现实 | 数据被打脸/预测落空 |

### 4. 传播链推演

预判这个话题的传播路径：

```
首爆：[哪个平台/哪个账号]
→ 第一波：[哪类 KOL 会转] → [什么角度]
→ 第二波：[媒体跟进] → [哪家媒体最可能报道]
→ 第三波：[普通用户转发] → [哪句话/哪个数据最容易被截图]
```

## 内容角度选择逻辑

根据话题类型选择最适合的切入角度：

| 话题类型 | 最佳切入角度 |
|---------|------------|
| AI / 技术替代 | 「普通人影响」「职业前景」「如何自救」 |
| 财富分化 / 巨头 | 「阶层分析」「财富机会」「商业逻辑」 |
| 政策变化 | 「谁受益谁受损」「历史对比」「行动建议」 |
| 社会矛盾 | 「情绪观点」「两方都有道理吗」「结构分析」 |
| 黑天鹅事件 | 「深度复盘」「为什么没人预警」「接下来会怎样」 |
| 消费/生活方式 | 「身份认同」「代际差异」「钱花在哪里」 |
