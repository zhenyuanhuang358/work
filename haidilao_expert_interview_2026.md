## Q1：今年一季度、4月及五一海底捞主品牌门店翻台率数据？客单价水平？同比情况？

**A：**

首先要说明一个结构性问题——海底捞是港股上市公司，不发布季度或月度经营简报，下一个正式数据窗口是2026年8月的中期业绩公告。所有当前分析都基于已知锚点做推算。

**已知数据：**

- **2025年全年翻台率**：3.9次/天（来源：海底捞2025年年报，2026-03-24），较2024年的4.1次/天下滑0.2次
- **2026年春节黄金周首日（2月15日）**：翻台率达5.2次/天，创近三年新高（来源：腾讯新闻，2026-02-15）——这是节日峰值，不代表季度均值
- **2025年全年客单价**：97.7元，同比-4.9%（-5元）

【分析预测】

**2026年Q1翻台率估算：**
一季度横跨春节旺季（2月）和传统淡季（1月、3月），综合估算如下：
- 1月：约3.5-3.7次（开年淡季，翻台承压）
- 2月：约4.5-4.8次（春节节效应，与峰值数据一致）
- 3月：约3.6-3.8次（节后回落）
- **Q1均值推算：约3.85-4.0次/天**，同比基本持平或微升（2025年Q1可参照全年均值推断为3.9-4.1区间）

**4月翻台率估算：**
4月是传统"回落月"——春节红利消退，五一消费窗口未开，历史上这个时段表现普遍偏弱。估算约3.5-3.8次/天，与2025年同期大概率持平或小幅下滑。

**五一2026（5月1-5日）估算：**
结合春节首日5.2次/天的峰值表现及2025年五一数据，预计黄金周日均翻台率4.8-5.5次，整个假期客流较平日高出约25-35%，五天合计营业额约7-9亿元（主品牌口径）。

**客单价展望：**
公司在2025年下半年开始主动推出"性价比套餐"并扩大外卖"下饭菜"产品线，部分消费分流至低客单价产品，到店客单价预计在95-100元区间窄幅波动，同比趋于持平或继续小幅下滑。2026年客单价"止跌"是观察信号，尚未见反转迹象。

## Q2：杨利娟从特海国际回归的具体情况？管理层架构有何调整？

**A：**

这是2026年上半年海底捞最重要的人事事件，有几个关键节点需要理清：

**时间线（已确认）：**

| 时间 | 事件 |
|------|------|
| 2022年3月 | 杨利娟接替张勇任海底捞CEO，张勇退居幕后 |
| 2024年6月 | 杨利娟卸任海底捞CEO，转任特海国际（港股01769）CEO，负责海外业务 |
| 2026年1月13日 | 张勇重新出任海底捞CEO（接替苟轶群），同时扶植4位年龄35-44岁内部女将进入董事会 |
| **2026年4月15日** | **特海国际公告杨利娟辞任CEO，回归海底捞，统筹推动"红石榴计划"多品牌孵化战略** |

来源：新浪财经2026-04-16、36氪2026-04、澎湃新闻2026-04、新浪财经2026-01-14

**当前管理层架构：**
- **CEO**：张勇（创始人，2026年1月13日复任）
- **多品牌战略负责人**：杨利娟（红石榴计划总操盘，2026年4月15日起）
- **特海国际CEO**：接替人选截至目前尚无港交所公告

**这次人事调动的逻辑：**

杨利娟在2022年任海底捞CEO期间主导了"啄木鸟计划"，关闭300+家亏损门店——这是海底捞历史上规模最大的一次业务收缩，也是她建立内部威望的核心战绩。现在把她从特海国际调回，核心任务是操盘红石榴计划的多品牌孵化。

背景是：2025年净利润同比下滑14%，20个子品牌中大多处于投入期，需要一个有"关店决断力"和"扩张执行力"双重基因的管理者来做分拣——哪些品牌加码，哪些砍掉。张勇负责战略定力，杨利娟负责落地执行，这个组合是目前最务实的配置。

放她离开特海国际，相当于默认海外业务放缓优先级——2026年的国内多品牌战略是更大的机会。

大排档是目前子品牌里我最看好规模化潜力的一个，但核心问题是：首店效应退去后的稳态数据尚未完整形成（2025年12月广州首店，截至目前仅9家）。

- 三年目标：管理层表示潜力达到500家（来源：东方财富财富号，2026-04-02）

来源：东方财富财富号2026-04-02、CBNData、餐饮人必读2025-12-31

【分析预测：单店经济模型估算】

以下为行业推断，非公司官方披露数据：

| 项目 | 估算参数 | 说明 |
|------|--------|------|
| 门店面积 | 400-700平方米 | 中大店型，参考首店规模 |
| 座位数 | 120-150座 | 按中等面积估算 |

## Q6：如鮨寿司的单店经济模型与扩张进展？

**A：**

如鮨是红石榴计划里最让我意外的一个品牌——一个火锅公司做平价寿司，逻辑是通的，但执行难度不低。首店数据出乎预料地亮眼。

**已知数据（杭州龙湖金沙天街首店）：**
- 门店面积：500+平方米
- 客单价：约88元，主力价位9.9元/件和15元/件
- 翻台率：工作日约5次，周末高达8次/天
- 日营业额估算：工作日约4万元，周末约8万元（来源：虎嗅AI简报）
- 扩张意向：店长透露规划向北京、上海、杭州扩张
- 门店总数：未单独披露；截至2025年6月，红石榴旗下14个品牌共126家门店

来源：虎嗅AI简报；36氪2026年，《海底捞入局，寿司郎、滨寿司加速扩张，百亿日料市场回暖了？》

【分析预测：单店经济模型估算】

以下为行业推断，非公司官方披露数据：

| 项目 | 估算参数 |
|------|--------|
| 日均翻台率 | 5-6次（综合工作日+周末均值） |
| 客单价 | 88元 |
| 日均营业额 | 约4-6万元 |
| 月均营业额 | 约120-180万元 |
| 食材成本率 | 35-42%（三文鱼、北极甜虾等日料食材成本较高） |
| 人工成本率 | 20-25%（寿司需一定技能工，略高于火锅） |
| 租金成本率 | 12-18%（商场铺位） |
| **营业利润率** | **估算8-15%** |
| 投资回收期 | 24-36个月 |

**如鮨的核心商业逻辑**是"日料快餐化"——把人均150元以上的日料体验压到88元，用高翻台弥补低单价。这个逻辑在一线城市高密度消费场景里有效，但：
- 下沉空间不如大排档（日料在低线城市认知度和消费习惯不足）
- 对新鲜食材的依赖使冷链要求更高，选址灵活度低于火锅类品牌

2026年如鮨大概率保持"精品扩张"节奏，率先覆盖北京/上海/杭州/成都等日料消费成熟城市，不具备大排档那种"500家"的规模化逻辑。

## Q7：1-4月外卖业务数据与全年预期？

**A：**

这是一个有结构性数据缺口的话题——海底捞不发月度数据，1-4月外卖专项数据无官方来源。

**2025年全年基准（已确认）：**
- 外卖收入：26.58亿元，同比+111.9%
- 占总营收比例：6.1%（2024年为2.9%）
- 全国外卖网点：超1200个
- 核心驱动产品："下饭菜/拌饭"，切入工作餐市场，客单价约25元

来源：第一财经2026-03-24；澎湃新闻2026-03-30；新浪财经2026-03-24

【分析预测：2026年外卖业务展望】

**增速节奏：**
2025年是外卖爆发年（+111.9%），2026年不可能维持翻倍增速。但绝对规模仍在扩张，增速预计降至25-45%区间：

| 情景 | 增速假设 | 全年外卖收入估算 |
|------|--------|--------------|
| 保守 | +20-25% | 31-33亿元 |
| 中性 | +30-40% | 35-37亿元 |
| 乐观 | +40-50% | 37-40亿元 |

**驱动因素：**
- 外卖网点持续扩张（目标2000+个网点）
- "下饭菜"产品线扩充，追加菜品SKU
- 外卖与到店的组合营销（"外卖消费积分兑换到店权益"类模式）

**战略意义不只是收入：** 外卖网点覆盖正在成为海底捞的"数字化毛细血管"——每个网点都是数据采集点和潜在到店转化漏斗。用25元客单价获取工作餐用户，再引流至客单价98元的到店消费，是一个值得关注的LTV逻辑。

## Q8：加盟业务2026年提速计划？

**A：**

加盟是海底捞2025年才正式落地的新业务模式，目前还在"验证跑道"阶段。

**已知数据（截至2025年末）：**
- 加盟门店：79家（自营1304家），占比5.7%
- 加盟申请积压：超2万份，70%来自三线及以下城市（含县级城市）
- 费用结构：**不收品牌加盟费**；加盟商自担资产+装修+运营全部成本，另付固定比例管理服务费；资金门槛约1000万元以上
- 官方态度：**明确表示不设具体加盟数量目标**，策略为"质量优先、下沉市场"

来源：证券时报；36氪《申请超2万份，已开出41家，加盟海底捞，你要准备多少钱？》；新浪财经2026-03-05

【分析预测：2026年加盟趋势】

**关键判断：** 供给端（合格加盟商筛选）是瓶颈，而非需求端（申请量充足）。

| 情景 | 年内新增加盟门店 | 年底加盟总数 | 条件 |
|------|--------------|-----------|------|
| 保守 | 40-50家 | 约120-130家 | 维持现有节奏，质量优先 |
| **中性** | **60-80家** | **约140-160家** | 开始提速，审核标准略降 |
| 乐观 | 100+家 | 180+家 | 管理层明确宣布提速 |

**值得关注的变量：**
杨利娟回归主导红石榴计划后，加盟可能不只限于主品牌。如果举高高（自助小火锅，客单价40-60元）和如鮨同步引入加盟模式，加盟数量的统计口径会扩大，实际加盟规模增速可能超预期。

**主要风险：**
1000万门槛筛选的加盟商多为本地资金方，未必有餐饮运营经验；海底捞主品牌的服务文化依赖长期培训和内部晋升文化，加盟体系下能否复制是核心挑战。这也是海底捞之前长期坚持纯直营的原因——现在"破禁"，说明增长压力已经大于品控风险的担忧。

## Q9：竞争格局——巴奴和小火锅赛道的动态？

**A：**

2026年火锅行业竞争格局用一句话总结：**行业出清加速，强者更强，但增量来自存量切割而非市场扩张。**

**巴奴动态：**

- **IPO进程**：2025年6月向港交所递交上市申请（联席保荐人中金+招银国际），2025年12月更新材料，目前进程推进中（来源：澎湃新闻2025年、腾讯新闻2025-12-25）
- **门店规模**：145家直营门店，覆盖39个城市，78.6%在二线及以下城市
- **三年扩张计划**：2026/2027/2028年分别新开约52/61/64家，三年合计+177家，总数翻倍
- **核心经营数据对比（2024年）**：

## Q10：2026年消费者趋势对海底捞的影响？

**A：**

2026年消费者趋势核心词是：**分层加剧、情绪溢价、下沉提速。**

**趋势一：性价比主导但非"低价化"**

- 中端价位（50-200元）食品消费占比68.7%，高端仅12.3%
- 消费者"不是不消费，是不愿意为复杂买单"（来源：21经济网2026-01-07）

**对海底捞的影响：** 97.7元的客单价正处于中端区间上沿，本身不是问题。真正的问题是：海底捞传统的"繁复服务仪式感"（洗头、美甲、暖场等衍生服务）在2026年年轻客群里的认知价值是否还成立？调研数据显示，这些服务仍有品牌辨识度，但实际使用率持续下滑。

**趋势二：情绪价值驱动**

- 25岁以下群体最看重：创意菜品（7.34/10）、精致餐食（7.16/10）
- 情绪价值来源：餐后小惊喜57.6%、好友共聚47.6%、个性化服务42.9%（来源：钛媒体2026年）

**对海底捞的影响：** 正负参半。餐后小惊喜和个性化服务恰恰是海底捞的传统强项，护城河仍在。但"创意菜品"评分最高，而这是海底捞的明显短板——火锅底料和食材创新相对保守，这是巴奴的毛肚、朱光玉的特色锅底等竞争者切入的空间。

**趋势三：下沉市场加速**

- 三线及以下城市餐饮门店占比预计达65%，高开店率持续
- 加盟申请70%来自三线以下城市，印证下沉需求（来源：证券时报）

**对海底捞的影响：** 这是最大的机会。加盟模式和举高高自助小火锅是下沉的两条腿——加盟覆盖中等消费力的县城市场（客单价100元），举高高覆盖下沉市场（客单价40-60元）。2026年是这两条路验证期，结果决定海底捞下沉市场增长天花板。

来源：投资界2026-01；知乎《2026中国餐饮产业生态白皮书》

## Q11：展望2026年下半年，海底捞战略上的重要变化？

**A：**

结合目前掌握的信息，我判断2026年下半年有以下几个高概率战略动作：

**① 红石榴计划进入"大砍"期**

20个品牌中，预计3-5个在2026年底前被冻结扩张或退出——这是杨利娟回归的核心任务。资源将向单店模型已经跑通的品牌集中：**大排档**（海鲜+火锅，翻台5.5-6次/天，已有潜力验证）和**举高高**（自助小火锅，50+家下沉覆盖）是最可能拿到内部资源倾斜的两个品牌。

**② 加盟提速配合"质量红线"**

2万+积压申请给了充足的筛选空间。预计下半年加盟月均开店从5-7家提速至10-15家，但同步强化督导体系。下半年可能看到子品牌（举高高和如鮨）同步引入加盟模式的政策公告。

**③ 外卖从"试验"变"基础设施"**

如果上半年外卖收入继续保持25%以上增速，下半年外卖网点扩张会加速，目标大概率是2000+个网点。"下饭菜"产品线会扩充，外卖与到店的组合套餐（外卖消费积分兑换到店体验权益）是一个值得观察的新营销形式。

**④ 张勇策略：不打价格战，打速度战**

张勇2026年1月回归CEO，核心判断不是防守而是反攻——但反攻的武器是多品牌的扩张速度，而非与对手打价格战。下半年大概率有1-2个新子品牌宣布孵化，覆盖目前尚未布局的赛道（粉面类、轻食类是可能方向）。

**⑤ 特海国际面临战略重估**

杨利娟离任后，特海国际（海外业务）的战略会被重新审视。全球经济放缓背景下，东南亚市场增长相对稳健，但欧美门店扩张动力不足。预计2026年下半年出现"聚焦东南亚重点市场、缩减其他市场投入"的方向性调整。

**最重要的观察指标：**

2026年中报（预计8月发布）的翻台率数据是最关键的信号——如果主品牌翻台率仍无法回到4次/天，市场对整个多品牌故事的耐心会明显缩减，海底捞的估值逻辑将从"修复主品牌"切换为"押注子品牌能否撑起增长"，这是两个截然不同的投资框架。

张勇+杨利娟的组合是目前执行层面最强的配置，但2026年是"做"比"说"重要的一年——战略路径清晰了，关键是交付。
