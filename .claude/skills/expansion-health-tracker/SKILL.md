# 连锁餐企扩张健康度早期信号追踪

> 卖方只能描述数字，你能解释它从哪来、往哪去。

---

## 核心命题

任何一家连锁餐企的「虚胖扩张」，在财报数字恶化之前 **1-2 个季度**，已经在三个层面留下可观测信号：

```
扩张决策层（T-2 季度）→ 单店质量层（T-1 季度）→ 财务传导层（同步）
      领先指标                  同步指标                  验证指标
```

**这是因果链，不是平行分析角度。**

定价缺口永远在财报尚未定价的前两层。第三层（财务）是已经发生的事，价值在于用前两层解释为什么出现这个财务结果——而不是从第三层倒推第一层。

---

## 适用范围

- 连锁餐饮（火锅 / 快餐 / 茶饮 / 烘焙 / 正餐）及线下消费品连锁
- 上市公司（A股 / 港股）和非上市规模连锁（门店 >50 家）
- 最适用时点：扩张加速期、IPO 前后 12 个月、战略转型期

**不适用**：门店数 <30 家的早期品牌（信号量不足）；纯线上品牌。

## 工作流（五个 Phase，顺序不可颠倒）

### Phase 0：目标设置

收到品牌名后立即执行，**不需要等待用户确认**：

1. 识别：品牌名 + 上市状态（A股 / 港股 / 非上市）+ 品类（火锅 / 快餐 / etc.）
2. 加载 `references/baseline-library.md`，读取对应品类的失序期阈值参数
3. 确定分析时间窗口（默认：最近 3 个完整财年 + 最新半年/季报）
4. 确定蚕食半径初始值（按品牌客单价定档，见 baseline-library.md）

```
✅ Phase 0 输出（直接输出，不暂停）
目标：[品牌名]（[上市状态]·[品类]·客单价约¥[X]）
时间窗口：[YYYY]–[YYYY]
蚕食半径初始值：[X] 公里（[客单价档位]）
基准锚点：[失序期参照，如"参照海底捞2020年：年净增>300家为失序信号"]
```

---

### Phase 1：扩张决策层（领先，T-2）

加载 `references/layer1-expansion-decision.md`，运行三个信号：

| 信号 | 核心问题 | 领先幅度 |
|------|---------|--------|
| 选址梯度 | 新开门店的城市层级结构是否在下沉加速？ | T-2 |
| 同城蚕食半径 | 新店距最近存量店的距离是否低于保护半径？ | T-2 |
| 扩张速率节奏 | 扩张速度是否超过历史均值 150%？ | T-2 |

---

### Phase 2：单店质量层（同步，T-1）

加载 `references/layer2-store-quality.md`，运行三个信号：

| 信号 | 核心问题 | 领先幅度 |
|------|---------|--------|
| 新店爬坡曲线 | 近 2 年新店评分曲线是否低于健康期基准？ | T-1 |
| 差评关键词结构 | 服务类 / 性价比类差评占比是否在上升？ | T-1 |
| 新老店评分剪刀差 | 新店评分是否系统性低于 5 年以上老店？ | T-1 |

**反直觉解读必须应用**（不能只描述现象）：
- 服务类差评↑ ≠ 服务变差，= 扩张过快、培训跟不上
- 性价比类差评↑ ≠ 价格太贵，= 消费者体验/期望失衡，需求预警
- 等位时间缩短 ≠ 效率改善，= 客流减少，门店冷清

---

### Phase 3：财务传导层（验证，同步）

加载 `references/layer3-financial.md`，运行三个信号：

| 信号 | 核心问题 |
|------|---------|
| 区域收入 / 门店剪刀差 | 区域门店增速是否持续高于该区域收入增速？ |
| 翻台率区域截面 | 哪些区域的翻台率已经在下滑？一线 vs 三线是否分化？ |
| 单店开设成本趋势 | 资本开支 ÷ 新开店数的年度趋势是否在上升？ |

上市公司加载年报数据；非上市公司标注`【数据缺口】`，用行业基准替代。

---

### Phase 4：定价缺口报告输出

加载 `references/output-template.md`，生成：
1. **三层信号汇总表**（当前读数 + 健康/危险判断 + 市场是否定价 + 领先财报幅度）
2. **一句话判断**（定价缺口方向 + 预计何时被市场定价）
3. **标的替换检查清单**（下一个分析标的的参数替换项）
4. （可选）HTML 报告，保存到 `reports/[slug]_expansion_[YYYY].html`

## 质量自检

```
□ 三层都有实际数据填入（不是占位符）？
□ 反直觉信号解读已应用于第二层（不只是描述现象）？
□ 一句话判断有明确方向（风险 / 机会）和时间预估？
□ 所有数据标注来源和日期？
□ baseline-library.md 的失序期参数已被引用作比较基准？
□ 客单价对应的蚕食半径档位已填入？
```
