# 社媒定性信号采集框架

调研提纲含"用户画像"/"消费者反馈"/"口碑"/"社媒表现"/"舆情"/"用户状态"时加载。
与 expansion-health-tracker 第二层（大众点评差评）配合使用，提供多平台交叉验证；也可独立用于品牌调研的用户状态分析。

核心问题：**消费者在说什么、说的最多的是什么、和上一个周期相比在变好还是变差？**

---

## 一、平台信号特征与适用场景

| 平台 | 信号特征 | 最适合分析 | 局限性 |
|------|---------|---------|--------|
| **小红书** | 图文体验记录，消费决策前参考 | 产品力感知、客单价感知、适合人群定位 | 正向内容偏多，差评相对少 |
| **微博** | 热点扩散，事件驱动 | 品牌危机、食安事件、营销效果 | 日常口碑信号噪声大 |
| **抖音** | 探店/开箱视频，互动量大 | 门店体验、出餐效率、品牌调性 | 内容娱乐化，口碑信号弱 |
| **黑猫投诉** | 正式投诉记录，有品牌回复 | 系统性服务问题、食安问题、加盟商纠纷 | 仅覆盖严重问题，冰山一角 |
| **大众点评** | 评分+文字评论最结构化 | 口味/服务/环境/性价比系统性追踪 | 见 expansion-tracker layer2 |

## 三、情绪量化方法

WebSearch 搜出来的是零散内容，需要以下方法转化为可量化信号：

### Step 1：采样规则

⚠️ **重要限制**：WebSearch 只返回搜索摘要片段，不是完整内容。情绪分布是相对趋势判断，不是精确计数——比绝对数字更有意义的是**与基准期的变化方向**。

- 每次 WebSearch 取前 10–15 条有效结果（排除广告、品牌官方账号、SEO软文、纯图片链接）
- "有效内容"定义：有实质体验描述的用户内容（如"上次去了，菜品……"），排除纯广告词、活动通知
- 时间范围：分两段——
  - **当期**：提纲指定时段，或最近完整季度（如 Q1 2025）
  - **基准期**：一年前同期（如 Q1 2024）；若提纲明确指定对比时段，按提纲为准
- 标记每条内容的：情绪倾向（正/负/中性）+ 核心关键词

### Step 2：分类框架

| 一级分类 | 二级分类 | 典型关键词 |
|---------|---------|---------|
| 产品力 | 口味 | 好吃/难吃/食材好/食材差/变味了 |
| 产品力 | 新鲜度 | 新鲜/不新鲜/预制/料理包 |
| 服务 | 态度 | 热情/冷淡/态度差/服务好 |
| 服务 | 效率 | 上菜快/等太久/排队 |
| 性价比 | 价格感知 | 值/不值/贵了/性价比高/低 |
| 性价比 | 分量 | 分量足/缩水/量少了 |
| 氛围 | 环境 | 装修/嘈杂/干净/整洁 |
| 危机 | 食安 | 食品安全/异物/变质 |

### Step 3：输出格式

```
[品牌名] 社媒情绪扫描 [时间段]

平台情绪分布（当期）：
- 小红书：正向[X]%  负向[Y]%  中性[Z]%（样本量N条）
- 黑猫投诉：[X]条（较去年同期 +/-Y%）

与基准期对比：
| 分类 | 当期占比 | 去年同期 | 变化 | 解读 |
|------|---------|---------|------|------|
| 产品力·口味 | X% | Y% | ↑/↓ | [说明] |
| 性价比感知 | X% | Y% | ↑/↓ | [说明] |
| 服务·效率 | X% | Y% | ↑/↓ | [说明] |

关键词热度变化：
上升词：[词A]、[词B]（→ 说明方向）
下降词：[词C]、[词D]（→ 说明方向）

平台特殊信号：
[黑猫投诉具体类型] / [微博热搜事件] / [小红书踩雷高赞笔记主题]
```

## 四、反直觉解读规则

与 expansion-health-tracker 一致，社媒信号同样不能只描述现象：

| 表面现象 | 真实解读 |
|---------|---------|
| 小红书正向内容↑ | 可能是品牌加大KOL投放，不等于真实体验变好。判断是否KOL投放：搜索结果中粉丝量 >1万的达人账号占比高（>50%）= 投放为主；普通用户账号为主 = 有机传播。需与黑猫/大众点评交叉验证 |
| 黑猫投诉数↑ | 可能是加盟店扩张带来的服务标准稀释，也可能是价格投诉（消费者体验/期望失衡） |
| 搜索量↑ | 可以是正向爆发（网红/出圈），也可以是危机扩散——需看内容情绪方向 |
| 差评中"以前很好"表述↑ | 老客户流失信号；品牌在消耗存量忠诚度，不是在吸引新客 |
| "排队减少"相关内容↑ | 客流下降，不是运营效率提升（同 expansion-tracker 等位时间信号） |

## 五、与其他分析维度的整合

| 社媒信号 | 联动分析维度 |
|---------|------------|
| 口味类差评↑ | → expansion-tracker Layer 2（差评关键词） + Layer 1（供应链覆盖半径） |
| 服务类差评↑ | → expansion-tracker Layer 2（培训跟不上扩张速度） |
| 性价比差评↑ | → expansion-tracker Layer 2（消费者期望/体验失衡，需求预警） |
| 食安事件 | → 黑猫投诉量 + 监管部门处罚公告（市场监管总局） |
| KOL投放密度↑ | → 结合门店客流判断：真实需求驱动 vs 营销驱动 |
