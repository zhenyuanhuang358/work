## 长隆集团 Chimelong · 酒店 MICE 会议中心

- **研究日期**：2026-06-15（v1）/ 2026-06-18（v2 定价专项） ｜ **报告**：`reports/chimelong_mice_Research_Report.html` + `reports/chimelong_mice_v2_Research_Report.html` ｜ **综合置信** 5.8→6.0/10
- **场馆硬件（高置信）**：广州长隆国际会展中心 6,000㎡无柱宴会厅（12m层高，3,000人）+ 39间多功能厅 + 1,500间客房（+熊猫酒店1,500间）。珠海横琴湾 3,000㎡海洋殿 + 1,200㎡水晶殿（合计3,500人）+ 26间多功能厅 + 25,000㎡海景草坪。
- **客户来源（中置信）**：广州长隆 本地45-55%；横琴湾 外地50%+、港澳20-30%（澳门口岸5分钟）。
- **四要素权重 推断（中置信）**：特色IP 30 / 配套住宿 28 / 价格 25 / 空间规模 17（百分制，度假型MICE）。
- **场租 行业基准推算（高置信-基准/低置信-长隆实际）**：3,000㎡大厅 淡季15-22万、旺季22-35万/天；中型宴会厅 2-10万/天；小型会议室 3千-1.2万/天。附加：搭建管理费1-5千、超时费场地费10-20%/小时、押金20-50%。
- **IP溢价（中置信）**：综合打包 +15%~35% vs 城区五星；关键是"隐性折抵"——企业会扣除乐园门票（人均200-400元/天）后重算性价比。
- **营收结构（中置信）**：客房30-40% / 餐饮28-38% / 场租12-20% / 设备搭建8-15% / 增值5-10%。
- **代理返点（低置信）**：行业惯例8-15%，主流约10%；长隆品牌强势相对克制（8-12%）。直销占35-50%、PCO 25-35%。
- **爬坡规律（中置信·行业）**：盈亏平衡3-5年（重资产可至7年）；第1年出租率30-50%、第3年60-70%、成熟期70-80%。
- **关键提醒**：场租"标价"≠"成交价"（多并入一价全包或被餐饮最低消费抵扣，净价可低30-50%）；长隆私营不披露，所有运营数字为行业基准推算。
