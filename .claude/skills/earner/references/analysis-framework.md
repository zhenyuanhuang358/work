# 五维度分析框架

Earner 的核心分析引擎。每次生成报告前，按顺序执行以下五个维度。

---

## 维度 1：语气评分（Tone Score）

**评分范围**：0–100，越高越乐观。

**评分方式**：

| 分段 | 标签 | 典型信号 |
|------|------|---------|
| 80–100 | 强势乐观 | "record"/"超预期"/"加速增长"/"上调指引" |
| 60–79 | 审慎乐观 | "稳健"/"符合预期"/"维持指引"/"看好长期" |
| 40–59 | 中性 | 中规中矩，无明确方向词 |
| 20–39 | 谨慎防御 | "挑战"/"承压"/"不确定性"/"暂缓扩张" |
| 0–19 | 熊市信号 | "下调指引"/"重组"/"减值"/"流动性压力" |

**三子维度**（各 0–100，综合加权得总分）：

| 子维度 | 权重 | 考察内容 |
|--------|------|---------|
| 前瞻指引完整度 | 35% | 是否给出量化 guidance，指引精确度 |
| 数据透明度 | 35% | 关键 KPI 是否全部披露（停披为负信号） |
| 战略清晰度 | 30% | 战略表述是否具体可执行 |

**无 transcript 时**：基于年报/公告措辞 + 已停止披露的指标判断，在报告中注明「基于书面报告推断」。

---

## 维度 2：关键数字（Key Numbers）

**必须提取的指标**（按优先级）：

```
营收（总额 + YoY%）
净利润 / 归母净利润（+ YoY%）
毛利率（+ 变动 bps）
EPS（实际 vs 共识预期）→ 超预期/符合/低于
同店销售增长率 SSS（餐饮/零售必选）
经营现金流（可选）
分部收入拆分（≥2 个分部时必填）
门店数 / 期末 KPI（餐饮品牌必填）
```

**EPS 计算规则**：
- 美股：EPS = 归母净利润 / 摊薄加权平均股数（年报财务报表注释）
- 港股：EPS（港仙）= 归母净利润（港元）/ 股数 × 100
- A股：EPS（元）= 归母净利润 / 总股本
- 若股数未披露：标注 `badge-gap` 并用市值倒推估算，注明「估算」

**Badge 规则**：
- 年报/官方公告直接披露 → `badge-official`（绿底）
- 分析师共识/市场预期 → `badge-predict`（琥珀底）
- 公司停止披露或未找到 → `badge-gap`（红底）

## 维度 3：管理层预期差（Guidance Gap）

**结构**：

1. **指引原话**（verbatim，引号内，不加工）
2. **市场隐含预期**（搜索"consensus estimate"或"analyst forecast"）
3. **差距判断**：
   - 实际/指引 > 市场预期 +5% → 正向预期差
   - 实际/指引 在 ±5% 内 → 符合预期
   - 实际/指引 < 市场预期 -5% → 负向预期差

**港股/A股无 transcript 时**：
- 指引原话 = 年报中管理层报告/展望章节的直接引用
- 市场预期 = WebSearch `[公司名] analyst forecast [年份]` 或路透/彭博共识数据
- 如无市场预期数据：标注 `badge-gap`，跳过差距判断，仅呈现管理层表述

## 维度 4：三情景目标价（Three-Scenario Target Price）

**步骤**：

**Step A：确定估值方法**
- 盈利稳定 / 利润率正常 → P/E 为主，P/S 为辅
- 高成长 / 亏损期 → P/S 为主，EV/Revenue 为辅
- 餐饮/消费品低利润 → P/S 为主（净利率极低时 P/E 失真）

**Step B：确定基准倍数**
- 搜索 `[行业] comparable P/S P/E multiple 2026` 获取同业水平
- 基准情景 = 当前隐含倍数（现价 / 当期收入或利润 per share）
- 熊市情景 = 基准下调 30–50%
- 牛市情景 = 基准上调 40–70%

**Step C：计算目标价**

```
市值 = 估值倍数 × 营收（P/S）或 净利润（P/E）
每股价值 = 市值 / 股本（注明货币）
```

**货币规则**：
- 港股：目标价以 HKD 表示，计算过程注明 RMB/HKD 换算率（≈1 RMB = 1.1 HKD）
- A股：目标价以 RMB 表示
- 美股：目标价以 USD 表示

**输出格式**：熊/基准/牛三张卡 + SVG 价格区间图 + 关键假设对照表（4行）

## 维度 5：核心主题 + 风险矩阵

**主题 chips**：
- 绿色（正向）：超预期增长、毛利扩张、市场份额提升
- 琥珀色（中性/观察）：指引不确定、成本压力、扩张放缓
- 红色（负向）：同店销售下滑、利润率收缩、管理层披露减少

**风险矩阵**（4–6行）：
- 列：风险项 / 概率（高/中/低）/ 影响（高/中/低）/ 监测信号
- 概率 × 影响 = 双高 → 在裁决框中必须提及

## 五维度自检

```
□ 语气评分有三子维度分解，有具体引用词句？
□ 关键数字全部有 badge 标注来源？
□ EPS 计算有股本来源说明？
□ 管理层预期差有指引原话引用（或注明无法获取）？
□ 目标价有明确倍数假设和货币单位？
□ 主题 chips ≥ 6 个，风险矩阵 ≥ 4 行？
```
