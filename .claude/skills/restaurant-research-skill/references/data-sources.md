## 搜索技巧

**找一个品牌的全貌**：
```
[品牌名] site:36kr.com OR site:late.cn OR site:huxiu.com 2025 OR 2026
```

**找业绩数据**：
```
[品牌名] 业绩 财报 营收 净利润 [年份]
```

**找非上市品牌门店数**：
```
[品牌名] 门店数量 扩张 [年份]
```

**找竞争格局**：
```
[品类] 市场格局 连锁 品牌 市场份额 [年份]
```
