# 巴菲特评分卡（Berkshire Score）

满分 100 分。每个维度的分数基于可查数据，不允许主观感觉填分。

---

## 维度一：Business Quality（30分）

### 1A. 护城河强度（0-15分）

| 护城河评分（moat-framework.md） | 得分 |
|-------------------------------|------|
| 0-2（无护城河） | 0（终止分析） |
| 3-4（窄护城河） | 5 |
| 5-6（中等） | 8 |
| 7-8（宽护城河） | 12 |
| 9-10（卓越护城河） | 15 |

### 1B. 收入稳定性（0-8分）

| 条件 | 得分 |
|------|------|
| 近3年营收 CAGR > 20% 且无负增长年份 | 8 |
| 近3年营收 CAGR 10-20%，最多1年负增长 | 6 |
| 近3年营收 CAGR 5-10%，无负增长年份 | 4 |
| 近3年营收 CAGR 5-10% 但含1年负增长，或 CAGR < 5% 无负增长 | 2 |
| 增速不稳定（多年波动）、CAGR < 5% 含负增长年，或多年持续下滑 | 0 |

> **边界处理**：严格按数字查表，不插值。CAGR 接近档位边界时取实际数字所在档，不向上取整。

### 1C. 定价权证据（0-7分）

| 条件 | 得分 |
|------|------|
| 毛利率近3年稳定 > 50% | 7 |
| 毛利率近3年稳定 35-50% | 5 |
| 毛利率 20-35% 且无明显压缩 | 3 |
| 毛利率持续压缩或 < 20% | 1 |

## 维度二：Financial Health（30分）

### 2A. 资本回报率（0-12分）

| 条件 | 得分 |
|------|------|
| ROE 3年均值 > 25% 且无大量杠杆驱动 | 12 |
| ROE 3年均值 18-25% | 9 |
| ROE 3年均值 12-18% | 6 |
| ROE 3年均值 8-12% | 3 |
| ROE < 8%（含负值）或主要靠债务杠杆 | 0 |

> **杠杆检验**：若负债率 > 40%，ROE 需用 ROIC 验证。ROIC < 12% 则降2档。

> **收购商誉扭曲检验**：若公司近3年完成大型收购（收购价 > 当年营收 50%），商誉膨胀股东权益分母，导致 GAAP ROE 虚低。此时必须同时计算调整 ROE = GAAP 净利润 / (股东权益 - 商誉)，并在报告 §二 中注明两个数字。评分仍按 GAAP ROE，但须标注「商誉扭曲，非运营质量原因」，防止读者误判业务健康度。

> **大额一次性减值扭曲检验**：若近3年内某年发生大额非现金资产减值（减值金额 > 当年营收 10%），GAAP 净利润被压低，导致该年 ROE 大幅失真（甚至为负）。须在报告 §二 中注明减值年份、减值金额，并计算剔除减值后的调整净利润/调整 ROE 供参考。评分仍按 GAAP ROE 三年均值执行，但须标注「含减值扭曲年份，GAAP ROE 失真，非运营质量原因」。三年均值若因该年被大幅拖低，评分结果须附注说明。

### 2B. FCF 质量（0-10分）

| 条件 | 得分 |
|------|------|
| FCF 连续3年为正 + FCF/净利润 > 80% | 10 |
| FCF 连续3年为正 + FCF/净利润 60-80% | 7 |
| FCF 2年正1年负，整体趋势向好 | 5 |
| FCF 不稳定或持续低于净利润 50% | 2 |
| FCF 多年为负 | 0 |

> **⚠️ 收购摊销扭曲**：大型并购后，经营现金流会加回收购无形资产摊销（非现金），导致 FCF 远大于 GAAP 净利润，FCF/净利润比率虚高。此时 10 分不代表盈利质量优秀——评分正常执行，但须在报告 §二 中标注：「FCF/NI 比率因收购摊销加回虚高，反映的是现金管理质量而非盈利转化率」，并注明剔除摊销加回后的调整 FCF/NI。

> **净利润 ≤ 0 时的处理**：FCF/净利润比率无意义（负值或无穷大），跳过比率档次检验。改用 FCF 绝对值趋势：① FCF 为正但净利润为负（减值/摊销等非现金费用驱动）→ 按 FCF 正负与连续性对应上方档次评分（忽略比率条件），并在报告 §二 标注「净利润含非现金亏损，FCF/NI 比率失效，按 FCF 绝对值评分」；② FCF 与净利润均为负 → 直接 0 分。

### 2C. 资产负债表健康（0-8分）

| 条件 | 得分 |
|------|------|
| 净现金（现金 > 总有息负债） | 8 |
| 负债率 < 20% 且 EBIT/利息 > 10x | 6 |
| 负债率 20-40% 且 EBIT/利息 > 5x | 4 |
| 负债率 40-60% 或 EBIT/利息 3-5x | 2 |
| 负债率 > 60% 或 EBIT/利息 < 3x | 0 |

> **负 EBIT 处理**：若运营利润为负（EBIT < 0），利息覆盖率 = 负值，直接取 0 分。在报告 §二 中注明「当前运营亏损，利息覆盖为负」，并说明预期盈利转折时间线（若管理层有指引）。此情况不与「净现金」条件并用——即使公司有净现金，EBIT < 0 仍得 0 分，因亏损本身代表持续偿债能力风险。

## 维度三：Management Quality（25分）

### 3A. 资本分配记录（0-12分）

基于近3年实际决策（不是管理层说的，是做的）。从下表中逐项判断，加总后截断在 [0, 12]：

**基础分：6分**（代表"没有明显失误，也没有明显亮点"）

| 行为（有证据才加/减） | 调整 |
|---------------------|------|
| 在低估值时回购（当时 PB < 2 或 PE < 15，有公告可查） | +4 |
| 分红连续增长 3 年以上 | +2 |
| 投资有明确 ROIC > 20% 的内部项目（有数据支撑） | +2 |
| 高溢价并购后商誉减值（有减值公告） | -4 |
| 股权激励连续 3 年稀释 > 3%/年 | -3 |
| 举债并购且 EBIT/利息 < 3x（可查财报） | -4 |

最终得分 = max(0, min(12, 6 + 加减合计))

> **CEO 换任处理**：若近3年内发生 CEO 替换，前任造成的负面决策（减值、过度 Capex）正常扣分。若新 CEO 有明确纠正行为（缩减 Capex、停止低效扩张、个人公开市场购股），可酌情 +1 到 +2 分（需有可查证据，新 CEO 在任须超过 1 年）。新 CEO 在任 < 1 年的行为不计分，在报告 §三 中注明「管理层任期不足，历史样本偏少」。

### 3B. 管理层与股东利益一致性（0-8分）

| 条件 | 得分 |
|------|------|
| 创始人仍在主导 + 持股比例 > 20% | 8 |
| 创始人主导 + 持股 10-20% | 6 |
| 职业经理人 + 大量持股（薪酬激励结构良好）| 5 |
| 职业经理人 + 少量持股 | 3 |
| 股权分散 + 无明确绑定激励 | 1 |

### 3C. 沟通诚实度（0-5分）

基于近2年财报、电话会议、投资者日：

| 条件 | 得分 |
|------|------|
| 主动披露风险和失误，承认错误 | 5 |
| 中性，没有明显粉饰，也没有深度披露 | 3 |
| 措辞模糊，回避关键指标（如毛利率分部数据） | 1 |
| 有误导性陈述或数据前后矛盾 | 0 |

## 维度四：Price vs Value（15分）

### 4A. 安全边际（0-10分）

| 当前安全边际 | 得分 |
|------------|------|
| ≥ 40% | 10 |
| 30-39% | 8 |
| 20-29% | 5 |
| 10-19% | 3 |
| < 10% | 0 |

> **路径 B 安全边际调整**：当估值使用路径 B（正态化 FCF）时，正态化 FCF 可信度低于标准 DCF，有效买入门槛为 MoS ≥ 40%（非路径 A 的 30%）。§4A 评分表得分不变，但 30-39% 的安全边际在路径 B 下不应视为买入信号——须在报告中注明「路径 B 下观察，非买入」。

### 4B. 相对估值参考（0-5分）

与同行 EV/FCF 对比（仅作辅助，不能覆盖 DCF 结论）：

| 条件 | 得分 |
|------|------|
| EV/FCF 低于同行均值 20%+ | 5 |
| EV/FCF 与同行持平 | 3 |
| EV/FCF 高于同行均值 20%+ | 1 |

> **EV/FCF 无效处理**：若公司 FCF 为负导致 EV/FCF 无意义，按顺序替代：① 改用 EV/EBITDA 与同行对比（须在报告中标注用的是 EV/EBITDA）；② 若 EBITDA 也为负，改用 EV/Sales；③ 若无任何有效对比指标，§4B 得 0 分，在报告中注明「FCF/EBITDA 均为负，相对估值无有效锚点」。

## 评分结果解读

| 总分 | 结论 | 说明 |
|------|------|------|
| 85-100 | **强力买入区域** | 极罕见；巴菲特式机会，核心仓位 |
| 70-84 | **等待买入** | 质量优异，等安全边际达标后建仓 |
| 55-69 | **观察名单** | 有投资价值，但护城河或估值存在明显缺陷 |
| 40-54 | **谨慎** | 部分优点，但关键指标未达标，风险与收益不匹配 |
| < 40 | **排除** | 不符合巴菲特标准，资源应分配给更好机会 |

## 自动降级规则（覆盖总分，分两档）

### 直接「排除」（无论总分多高，终止分析）
- 近12个月发现财务造假或重大信息披露违规
- 护城河评分 ≤ 2（无任何可辨识护城河，商业模式不成立）

### 强制降至「谨慎」（不得给出买入建议）
- 护城河评分 3（窄护城河边缘，需单独说明不确定性）
- 护城河评分因侵蚀信号从 4 降至 3 → 同样触发（级联效应）；若原评分为 3，侵蚀后降至 2 → 升级触发「直接排除」
- 负债率 > 70% 且 EBIT/利息 < 2x
- FCF 连续3年为负（尚未证明可持续盈利能力）

在报告中用醒目方式注明触发了哪条规则及原因。
