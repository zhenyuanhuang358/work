# 核心指标定义

报告内口径统一参考，避免同一指标在不同品牌间因定义不同产生误导。

---

| 指标 | 标准定义 | 口径说明 |
|------|---------|---------|
| **同店销售（SSS）** | 开业满N个月门店的销售额同比变化 | 各公司口径不同：海底捞用12个月，部分品牌用18个月；**报告中标注"SSS（[公司名]口径：[N]个月）"**；多品牌对比时需对齐到同一口径后再比较，否则无可比性 |
| **翻台率** | 每张桌子每天接待的批次数 | 通常指餐厅层面全天；午市/晚市分开披露时需注明；部分公司用"翻座率" |
| **AUV（门店平均月销）** | 单店平均月度销售额 | 直营门店数据更可靠；加盟门店AUV需注明数据来源；**若公司披露混合口径（直营+加盟合并），标注"混合口径"并说明：加盟占比高→ AUV 被低估（因加盟计收特许费/供应链收入，而非门店全收入）** |
| **客单价** | 每位顾客平均消费金额 | 含税或不含税需注明；外卖客单价通常低于到店客单价 |
| **餐厅层面利润率** | 单店收入 - 食材 - 人力 - 租金 - 折旧 | 不含总部费用摊销；也叫"四壁利润率"或"restaurant-level margin" |
| **净增门店** | 当期新开门店数 - 关闭门店数 | 关闭门店数经常不被主动披露，需从前后期总数倒推 |
| **首年回收期** | 加盟商收回初始投资所需时间 | 管理层给出的通常为理论值（满产假设），实际往往更长；**健康区间：12–18 个月；>24 个月为警戒信号；>36 个月招商困难**。加盟商实际反馈可在餐饮论坛/创业者社区获取 |
| **外卖占比** | 外卖收入 / 总营收 | 部分公司将外卖含在门店营收内，需查财报注释确认口径 |
| **毛利率（餐厅层面）** | (营收 - 食材成本) / 营收 | 不同于餐厅层面利润率（后者还扣除人工和租金） |
| **SSS基数效应** | 去年同期SSS的高低对本期SSS的影响 | 去年同期SSS高 → 今年基数高 → 今年SSS容易偏低，需说明 |
| **加盟渗透率** | 加盟门店数 / 总门店数 | 反映品牌对加盟的依赖程度；高加盟率品牌财报营收仅含特许费和供应链收入 |

---

## 使用提示

- 报告中第一次出现某指标时，可在括号内简短注明口径
- 若两个数据来源的同一指标口径不同，需在报告中说明并取一个为准
- 中国餐饮公司的"翻台率"口径与美国快餐公司的"table turns"口径基本一致，可做跨境对比参考

## 利润率口径对照（分析时必须区分）

| 利润率类型 | 包含项目 | 典型值 | 用途 |
|---------|---------|-------|------|
| **餐厅层面利润率**（四壁利润） | 门店收入 - 食材 - 直接人工 - 门店租金 - 折旧 | 头部品牌 15–25% | 评估单店经营效率 |
| **集团净利率** | 餐厅层面利润 - 总部管理费 - 研发 - 利息 - 税 | 通常低于餐厅层面 5–10ppt | 评估投资回报 |

> 两者差距 >15ppt → 往往暗示总部费用膨胀或效率问题，需在报告中说明。
> 分析经营质量用**餐厅层面利润率**；分析投资价值用**集团净利率**。
