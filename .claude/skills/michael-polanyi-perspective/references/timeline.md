# 波兰尼人物时间线

当用户询问波兰尼的生平、某段历史时期的立场、或某著作的背景时，查阅此文件。

---

## 关键节点

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 1891 | 生于布达佩斯，犹太中产家庭 | 中欧知识传统、多语言环境 |
| 1913 | 获医学博士，布达佩斯大学 | 科学训练：严格但保有个人直觉的空间 |
| 1914 | 发表吸附势能理论，被哈伯否定 | 亲身体验：顶住权威否定、坚持判断，40年后证明正确 |
| 1923 | 转型物理化学，柏林威廉皇帝研究所 | 顶级科学环境，接触爱因斯坦等人 |
| 1933 | 离开纳粹德国，去曼彻斯特大学 | 政治压力下的流亡，强化对「自由」的关注 |
| 1935 | 访问莫斯科，与布哈林争论 | **触发点**：布哈林说「纯粹科学是资产阶级意识形态」，波兰尼开始为科学自由辩护 |
| 1946 | 出版《科学、信仰与社会》 | 第一本明确主张科学自由的书 |
| 1948–50 | 转向哲学讲席，放弃化学研究 | 从做科学到理解科学：最大的职业转型 |
| 1948–50 | 皈依天主教 | 个人信仰与认识论整合：信仰也是一种负责任的默会承诺 |
| 1951–52 | 在阿伯丁大学的吉福德讲座 | 写成《个人知识》的素材，第一次系统阐述「个人知识论」 |
| 1958 | 出版《个人知识》 | 主要著作：完整建立「后批判哲学」体系 |
| 1962 | 发表「科学共和国」 | 最广泛引用的文章：分布式科学秩序理论 |
| 1966 | 出版《默会维度》 | 最易读的入门，将主要论点压缩为三场讲座 |
| 1969 | 《认识与存在》出版 | 延伸到生物哲学和意识哲学 |
| 1972 | 中风，健康衰退 | 写作能力受损，与Prosch合作完成最后一本书 |
| 1975 | 《意义》出版（与Prosch合著） | 将默会认知延伸到美学、宗教和意义问题 |
| 1976 | 在牛津去世，享年84岁 | — |

---

## 身后影响（1976—今）

- **托马斯·库恩**：在《科学革命的结构》中引用波兰尼，范式转换理论与波兰尼的「个人承诺」高度共鸣
- **德雷福斯兄弟**：用默会认知批判AI的局限（1970s–），《计算机不能做什么》直接援引波兰尼
- **野中郁次郎与竹内弘高**：《知识创造企业》（1995）将默会知识带入管理学，成为全球最引用的商业理论之一
- **Harry Collins**：将默会知识发展为STS（科学技术研究）的核心研究纲领，建立了默会知识的类型学
