# 第一层：扩张决策层操作指南（T-2 领先指标）

核心问题：**管理层现在在做什么决策，是否已经失序？**

数据来源：窄门餐眼（开店记录）+ 极海（地图坐标）+ 公司公告

---

## 信号一：选址梯度

**拆解维度**：新开门店的城市层级占比，按年度统计，识别下沉加速拐点。

### 数据采集

```
窄门餐眼操作：
1. 搜索目标品牌 → 门店列表 → 按"开业时间"筛选每年新开店
2. 每家门店有"城市"字段，对照城市层级分类表（见 baseline-library.md）
3. 统计每年：一线 / 新一线 / 二线 / 三线及以下 各占新开门店的比例

WebSearch 补充（窄门餐眼不可访问时）：
[品牌名] 门店分布 城市 一线 二线 三线 [年份]
[品牌名] site:narrow-door.com OR 窄门餐眼 城市层级
```

### 计算方法

```
选址下沉指数 = 三线及以下新开占比（当年）- 三线及以下新开占比（历史均值）

正值且持续扩大 = 下沉加速信号
```

### 判断阈值

| 状态 | 判断标准 |
|------|---------|
| ✅ 健康 | 三线占比年变化 < 5ppt；城市层级分布与历史均值偏差 < 10ppt |
| ⚠️ 警示 | 三线占比单年突增 10–20ppt；或连续 2 年下沉趋势 |
| 🔴 危险 | 三线占比超历史均值 +20ppt 以上；**同年多层级同时加速下沉** |

### 你的判断介入点

> 三线城市客单价天花板更低，同品牌门店间人口密度支撑更弱。
> "三线占比突然超过 X%" 这个 X 值是你根据实际运营经验校准的阈值，不是行业公式。
> 卖方分析师只能数总数，无法判断梯度失序的临界点。

## 信号二：同城蚕食半径

**拆解维度**：同城新开门店与最近存量门店的直线距离分布，识别内部竞争。

### 数据采集

```
窄门餐眼 + 极海操作：
1. 窄门餐眼导出目标品牌全部门店列表（含城市、地址、开业时间）
2. 极海批量解析地址为坐标（经纬度）
3. Python 计算：每家新开店（近2年）与同城所有存量店（>2年）的距离，取最小值
4. 绘制距离分布直方图，观察近2年 vs 基准期（5年前）的分布变化

WebSearch 补充：
大众点评地图视图：搜索品牌名 → 切换"地图"视图 → 目测主要城市门店密度
重点城市（北京/上海/成都/深圳各做一个样本）
```

### 计算方法

```
蚕食压力指数 = 近2年新开店中，距最近存量店 < 保护半径 的比例

保护半径由客单价档位决定（见 baseline-library.md）：
¥150+ 人均：保护半径 5–8 公里
¥80–150 人均：保护半径 3–5 公里
¥50–80 人均：保护半径 2–3 公里
¥50 以下：保护半径 1–2 公里
```

### 判断阈值

| 状态 | 判断标准 |
|------|---------|
| ✅ 健康 | 新店距最近存量店 > 保护半径的比例 > 80% |
| ⚠️ 警示 | 20–40% 新店落在保护半径内 |
| 🔴 危险 | >40% 新店落在保护半径内；**大量新店在存量店蚕食半径内集中** |

### 你的判断介入点

> 你管连锁门店，知道同一商圈真实能承载多少家同品牌。
> 这个阈值是从实际运营中校准的经验值，卖方没有这个经验，只能数总门店数。
> 蚕食半径过小的后果比失序扩张更隐蔽：客流稀释慢慢发生，翻台率先跌，再出现在财报里。

## 信号三：扩张速率节奏

**拆解维度**：月均新开门店数的滚动 12 个月曲线，识别加速/减速拐点。

### 数据采集

```
窄门餐眼操作：
1. 导出品牌全部门店的"开业时间"列
2. 按月统计新开门店数
3. 计算滚动 12 个月均值曲线

WebSearch 补充：
[品牌名] [年份] 新开门店数 扩张计划
[品牌名] 年报 新开门店 关闭门店 净增
```

### 计算方法

```
扩张速率偏离度 = (当前12月均值 - 历史5年均值) / 历史5年均值 × 100%

与 baseline-library.md 中的"失序期特征"对比：
- 海底捞2020年：年净增530家 / 总门店1300家 ≈ 40% 净增率
- 失序信号：直营为主的品牌年净增率 > 25%（参见 restaurant-research-skill 行业基准）
```

### 判断阈值

| 状态 | 净增率（净增数 / 期初总门店） | 绝对速率偏离度 |
|------|----------------------|------------|
| ✅ 健康 | < 15% | < 50% |
| ⚠️ 警示 | 15–25% | 50–150% |
| 🔴 危险 | > 25% | > 150%（对照基准库"失序期"参数） |

### 你的判断介入点

> 对比 2020 年失序期的特征参数（见 baseline-library.md）提炼的"失序扩张特征库"。
> 这个基准库随每个项目累积，是长期资产。
> 核心判断：当前加速是管理层计划内的战略扩张，还是超出供应链/人才储备能力的失控？
